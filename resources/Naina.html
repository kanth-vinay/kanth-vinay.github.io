Power Series, Taylor & Maclaurin Series

1 What Is a Power Series?

A power series centered at \(a\) is an infinite sum:

\(\displaystyle \sum_{n=0}^{\infty} a_n (x - a)^n = a_0 + a_1(x-a) + a_2(x-a)^2 + \cdots\)
  • If \(a = 0\), it is a Maclaurin series.
  • Each \(a_n\) is a coefficient.
  • The value \(a\) is the center.
Key identification skill: \(\sum c_n(x+3)^n\) is centered at \(a = -3\), not \(+3\).

2 Convergence: Radius & Interval

Every power series converges on an interval centered at \(a\).

Radius of Convergence \(R\)

Use the Ratio Test: \(\displaystyle R = \lim_{n\to\infty}\left|\frac{a_n}{a_{n+1}}\right|\)
  • \(R = 0\) → converges only at \(x = a\)
  • \(R = \infty\) → converges for all \(x\)
  • Finite \(R\) → converges on \((a-R,\, a+R)\), must check endpoints separately

Interval of Convergence (IOC)

After finding \(R\), test \(x = a-R\) and \(x = a+R\) individually. Endpoints may be included [ ] or excluded ( ).

Example: \(\sum x^n/n\) has \(R=1\). At \(x=1\): \(\sum1/n\) diverges. At \(x=-1\): \(\sum(-1)^n/n\) converges. IOC = \([-1, 1)\).

3 The Six Essential Maclaurin Series

FunctionMaclaurin SeriesIOC
\(\dfrac{1}{1-x}\)\(\displaystyle\sum_{n=0}^{\infty} x^n\)\((-1,1)\)
\(e^x\)\(\displaystyle\sum_{n=0}^{\infty}\frac{x^n}{n!}\)\((-\infty,\infty)\)
\(\sin x\)\(\displaystyle\sum_{n=0}^{\infty}\frac{(-1)^n x^{2n+1}}{(2n+1)!}\)\((-\infty,\infty)\)
\(\cos x\)\(\displaystyle\sum_{n=0}^{\infty}\frac{(-1)^n x^{2n}}{(2n)!}\)\((-\infty,\infty)\)
\(\ln(1+x)\)\(\displaystyle\sum_{n=1}^{\infty}\frac{(-1)^{n+1} x^n}{n}\)\((-1,1]\)
\(\arctan x\)\(\displaystyle\sum_{n=0}^{\infty}\frac{(-1)^n x^{2n+1}}{2n+1}\)\([-1,1]\)
Common trap: \(\sin x\) and \(\arctan x\) look similar — both have odd powers and alternating signs. The difference: \(\sin x\) has factorials in the denominator, \(\arctan x\) does not.

4 Coefficients from Derivatives

\(\displaystyle a_n = \frac{f^{(n)}(a)}{n!}\)

For a Maclaurin series (\(a=0\)):

  • \(a_0 = f(0)\)
  • \(a_1 = f'(0)\)
  • \(a_2 = f''(0)/2!\)
  • \(a_3 = f'''(0)/3!\)
Example: If \(f'''(0) = 24\), then \(a_3 = 24/6 = 4\), not 24.

5 Taylor Polynomial vs. Taylor Series

Taylor Polynomial \(P_n(x)\)Taylor Series
TermsFinite (degree \(n\))Infinite
PurposeApproximationExact representation
ConvergenceAlways a polynomialMay converge on a limited interval
\(\displaystyle P_n(x) = \sum_{k=0}^{n}\frac{f^{(k)}(a)}{k!}(x-a)^k\)
\(P_3(x)\) for \(e^x\) at \(a=0\): \(1 + x + \frac{x^2}{2} + \frac{x^3}{6}\). This has exactly 4 terms (degree 0 through 3).

6 Differentiating & Integrating Series

You can differentiate and integrate a power series term by term within its interval of convergence.

Differentiation

\(\displaystyle \frac{d}{dx}\!\left[\sum_{n=0}^{\infty}a_n x^n\right] = \sum_{n=1}^{\infty} n\, a_n\, x^{n-1}\)

Integration

\(\displaystyle \int\!\left[\sum_{n=0}^{\infty}a_n x^n\right]dx = C + \sum_{n=0}^{\infty}\frac{a_n\, x^{n+1}}{n+1}\)
Key application: \(\displaystyle\int_0^x \frac{1}{1+t^2}\,dt = \sum_{n=0}^{\infty}\frac{(-1)^n x^{2n+1}}{2n+1} = \arctan x\)

7 Building New Series (Substitution)

Replace the variable in a known series to create a new one.

Substitution Examples

TargetStart withSubstitute
\(e^{-x^2}\)\(e^u = \sum u^n/n!\)\(u = -x^2\)
\(\dfrac{1}{1+x^2}\)\(\dfrac{1}{1-u}=\sum u^n\)\(u = -x^2\)
\(\ln(1+2x)\)\(\ln(1+u)\)\(u = 2x\)
\(\arctan(2x)\)\(\arctan u\)\(u = 2x\)
\(\displaystyle e^{-x^2} = \sum_{n=0}^{\infty}\frac{(-1)^n x^{2n}}{n!}\) — uses \(n!\), not \((2n)!\)
Watch out: \((-1)^n x^{2n}/(2n)!\) is \(\cos x\), not \(e^{-x^2}\). The factorial is the giveaway.

8 Multiplying / Dividing Series

To find the series for a product like \(x\cos x\), multiply directly:

\(x\cos x = x\!\left(1 - \dfrac{x^2}{2} + \dfrac{x^4}{24} - \cdots\right) = x - \dfrac{x^3}{2} + \dfrac{x^5}{24} - \cdots\)

To find \(\sin x / x\), divide each term:

\(\dfrac{\sin x}{x} = \dfrac{1}{x}\!\left(x - \dfrac{x^3}{6} + \dfrac{x^5}{120} - \cdots\right) = 1 - \dfrac{x^2}{6} + \dfrac{x^4}{120} - \cdots\)

9 Taylor Series at \(a \neq 0\)

To build a Taylor series centered at \(a\), use substitution on a known Maclaurin series.

\(\ln x\) at \(a=1\): let \(u = x - 1\), so \(\ln x = \ln(1+u) = \displaystyle\sum_{n=1}^{\infty}\frac{(-1)^{n+1}(x-1)^n}{n}\)
The series must use \((x-a)^n\), not \(x^n\). If you see \(x^n\) in a series for \(\ln x\) centered at 1, it is wrong.

10 Approximation & Error Bounds

Using a Polynomial to Estimate

\(e^{0.1} \approx P_2(0.1) = 1 + 0.1 + \dfrac{(0.1)^2}{2} = 1.105\)

Alternating Series Error Bound

For an alternating series, the error from using \(n\) terms is at most the absolute value of the first omitted term.

\(|\text{error}| \le |a_{n+1}|\) (the first term you didn't include)
Example: \(\cos(0.5) \approx P_4(0.5) = 1 - \frac{(0.5)^2}{2} + \frac{(0.5)^4}{24}\). Error \(\le \frac{(0.5)^6}{720} \approx 0.0000217\).

! Common Mistakes to Avoid

  • Confusing radius \(R\) with interval — \(R=3\) does not mean IOC is \([0,3]\).
  • Forgetting to divide by \(n!\) when going from \(f^{(n)}(0)\) to \(a_n\).
  • Mixing up \(\sin x\) and \(\arctan x\) — check for factorials.
  • Using \((2n)!\) instead of \(n!\) for \(e^{-x^2}\) (that gives \(\cos x\)).
  • Writing \(x^n\) instead of \((x-a)^n\) when the series is centered at \(a \neq 0\).
  • Using the last included term for the error bound instead of the first omitted term.
  • Forgetting that \(P_3(x)\) has terms from degree 0 through degree 3 (four terms total for \(e^x\)).
  • Assuming endpoint convergence without testing — always check both endpoints.
