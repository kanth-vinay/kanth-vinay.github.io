    Chi-Square ($\chi^2$) Tests

    Analyzing Categorical Data

    Goodness-of-Fit • Contingency Tables • Test of Independence • Homogeneity

    I. The Core Statistical Engine

    The Foundational Formula

    $\chi^2 = \sum \frac{(O - E)^2}{E}$

    This singular formula dictates all Chi-Square hypothesis testing. It explicitly measures the deviation between reality and theoretical expectation.

    • $O$ (Observed): The empirical frequency recorded in the data.
    • $E$ (Expected): The theoretical frequency anticipated if $H_0$ is true.
    • $\sum$ (Summation): The sum of this ratio across all distinct categories or cells.

    Pattern Recognition

    Consider the numerator: $(O - E)^2$.

    • If observations perfectly match expectations, $O - E = 0$, yielding $\chi^2 = 0$.
    • The squaring ensures all deviations (positive or negative) accumulate positively.
    • Dividing by $E$ standardizes the deviation (a difference of 10 matters more if we expected 5 vs. if we expected 500).
    Axiom: A significantly large $\chi^2$ statistic forces the rejection of the null hypothesis.

    II. Typology of $\chi^2$ Tests

    1. Goodness-of-Fit

    Purpose: Validates if a single categorical variable matches a hypothesized theoretical distribution.

    1 Variable
    1 Sample

    $E = n \cdot p$

    $df = k - 1$

    Ex: Are M&M colors distributed according to Mars Inc. claims?

    2. Independence

    Purpose: Evaluates if two categorical variables within a single sample are mathematically associated or strictly independent.

    2 Variables
    1 Sample

    $E = \frac{R_{Total} \cdot C_{Total}}{G_{Total}}$

    $df = (r - 1)(c - 1)$

    Ex: Is Hair Color independent of Eye Color?

    3. Homogeneity

    Purpose: Compares the distribution of a single categorical variable across multiple distinct populations.

    1 Variable
    2+ Samples

    $E = \frac{R_{Total} \cdot C_{Total}}{G_{Total}}$

    $df = (r - 1)(c - 1)$

    Ex: Do survey responses differ between blindfolded vs. sighted groups?

    III. Goodness-of-Fit: The Fair Die

    We roll a standard die $n=120$ times. Does empirical evidence support $H_0: P(X_i) = \frac{1}{6}$?

    Face ($k$) Observed ($O$) Expected ($E$) $O - E$ $(O - E)^2$ $\frac{(O - E)^2}{E}$
    118$120 \cdot (1/6) = 20$-24$4/20 = 0.20$
    22520525$25/20 = 1.25$
    31520-525$25/20 = 1.25$
    4222024$4/20 = 0.20$
    52820864$64/20 = 3.20$
    61220-864$64/20 = 3.20$
    $\sum$ (Test Statistic) $\chi^2 = 9.30$
    $df = k - 1 = 6 - 1 = 5$
    Critical Value ($\alpha=0.05, df=5$) $= 11.07$

    Conclusion: Since $\chi^2 (9.30) < \text{Crit} (11.07)$, we fail to reject $H_0$. There is insufficient evidence to claim the die is unfair.

    Computational Verification (RStudio)

    M&M Color Distribution Case

    Test if a sample of $n=712$ matches Mars Inc. 2008 published percentages.

    • $H_0$: Sample matches (Blue 24%, Orange 20%, Green 16%, Yellow 14%, Red 13%, Brown 13%).
    • $H_1$: The distribution differs.

    Observed Counts: 133, 133, 139, 103, 108, 96

    R Execution

    # Define Vectors
observed_mm <- c(133, 133, 139, 103, 108, 96)
expected_prop <- c(0.24, 0.20, 0.16, 0.14, 0.13, 0.13)

# Execute Goodness-of-Fit
chisq.test(observed_mm, p = expected_prop)
    Output:
    X-squared = 17.066, df = 5, p-value = 0.004377

    Conclusion: Reject $H_0$ ($p < 0.05$).

    IV. Contingency Tables & Independence

    Calculating Expected Counts across two variables: $E_{rc} = \frac{\text{Row } r \text{ Total} \times \text{Col } c \text{ Total}}{\text{Grand Total}}$

    Observed Matrix ($O$)

    Category ACategory BRow Totals
    Group 16040100
    Group 2201030
    Col Totals8050GT = 130
    $df = (r-1)(c-1) = (2-1)(2-1) = 1$

    Expected Matrix Derivation ($E$)

    $E_{1,1} = \frac{100 \times 80}{130} = 61.54$
    $E_{1,2} = \frac{100 \times 50}{130} = 38.46$
    $E_{2,1} = \frac{30 \times 80}{130} = 18.46$
    $E_{2,2} = \frac{30 \times 50}{130} = 11.54$

    $\chi^2 = \frac{(60-61.54)^2}{61.54} + ... = 0.433$

    $p = 0.51 \rightarrow$ Fail to Reject $H_0$

    Analytical Checkpoint

    Formula Summary

    Statistic $\chi^2 = \sum \frac{(O - E)^2}{E}$
    Expected (1-Var) $E = n \cdot p$
    Expected (2-Var) $E = \frac{RT \cdot CT}{GT}$
    $df$ (1-Var) $df = k - 1$
    $df$ (2-Var) $df = (r - 1)(c - 1)$

    R Syntax Execution

    Executing these models requires structured data inputs.

    # Goodness of Fit
chisq.test(vector, p = expected_props)

# Independence
chisq.test(contingency_matrix)
