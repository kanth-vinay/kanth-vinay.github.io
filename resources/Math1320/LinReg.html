14.1 The Equation of a Line

Definition 14.1: $y$-Intercept and Slope

For a linear equation $y = b_0 + b_1 x$, the number $b_0$ is called the $y$-intercept and the number $b_1$ is called the slope.

Graphical Interpretation of Slope (Key Fact 14.1)

Lines Upward

$b_1 > 0$

Lines Downward

$b_1 < 0$

Horizontal

$b_1 = 0$

Practice Sprint

Recall Check: The Math of a Line
Before proceeding with linear regression, answer the following basic review questions:

  1. What is the formula to calculate the slope $m$ (or $b_1$) between any two points $(x_1, y_1)$ and $(x_2, y_2)$?
  2. If a line is perfectly horizontal, what is its slope?
  3. If a line is perfectly vertical, what is its slope?

Solution

  1. $m = \frac{y_2 - y_1}{x_2 - x_1}$ (Rise over Run)
  2. Zero. A horizontal line has no rise, so the numerator is 0.
  3. Undefined. A vertical line has no run, leading to division by 0!

The Population Regression Model

Before computing sample statistics ($b_0$ and $b_1$), we must recognize the true population reality. We model each individual data point ($Y_i$) as a perfect linear component plus random unavoidable error ($\varepsilon_i$).

$$ \overset{\substack{\text{\small Dependent}\\\text{\small Variable}\\\downarrow}}{Y_i} = \underbrace{ \overset{\substack{\text{\small Population}\\\text{\small Y-Intercept}\\\downarrow}}{\beta_0} + \overset{\substack{\text{\small Population}\\\text{\small Slope Coeff}\\\downarrow}}{\beta_1} \,\, \overset{\substack{\text{\small Independent}\\\text{\small Variable}\\\downarrow}}{X_i} }_{\text{Linear component}} + \underbrace{ \overset{\substack{\text{\small Random}\\\text{\small Error Term}\\\downarrow}}{\varepsilon_i} }_{\text{Random Error component}} $$

The Least-Squares Criterion

Key Fact 14.2

The "Line of Best Fit" for a set of data points is theoretically defined as the one line that produces the smallest possible sum of squared errors ($SSE$).

Comparing Multiple Lines

Given 3 simple points: $(1, 3), (2, 1), (3, 5)$, which arbitrary line fits better mathematically?

  • Line A ($\hat{y}=-1+2x$): $e^2 = 4^2+4^2+0 = \mathbf{8}$
  • Line B ($\hat{y}=1+x$): $e^2 = 1^2+4^2+1^2 = \mathbf{6}$

Notation Used in Regression and Correlation

For a set of $n$ data points, computational formulas are utilized to bypass complicated fraction arithmetic. (Definition 14.4)

Quantity Defining Formula Computing Formula
$S_{xx}$ $\Sigma(x_i - \overline{x})^2$ $\Sigma x_i^2 - (\Sigma x_i)^2 / n$
$S_{xy}$ $\Sigma(x_i - \overline{x})(y_i - \overline{y})$ $\Sigma x_i y_i - (\Sigma x_i)(\Sigma y_i) / n$
$S_{yy}$ $\Sigma(y_i - \overline{y})^2$ $\Sigma y_i^2 - (\Sigma y_i)^2 / n$

The Regression Equation (Formula 14.1)

The explicit regression equation (Line of Best Fit) for a set of $n$ data points is geometrically given by $\hat{y} = b_0 + b_1 x$, where:

$b_1 = \frac{S_{xy}}{S_{xx}}$ and $b_0 = \overline{y} - b_1\overline{x} = \frac{1}{n}(\Sigma y_i - b_1\Sigma x_i)$

Applied to Orion Data (Table 14.6)

Using the dataset of 11 cars we extracted $\Sigma x = 58$, $\Sigma y = 975$, $\Sigma x^2 = 326$, $\Sigma xy = 4,732$.

$S_{xx} = 326 - 58^2/11 = 20.18$ and $S_{xy} = 4732 - (58\times 975)/11 = -408.9$

Resulting Equation: $\hat{y} = 195.47 - 20.26x$

Hazards and Best Practices

Extrapolation (Fig 14.12)

Using the regression equation to make predictions outside the known bounds of observed $x$-values is severely flawed extrapolation.

Example: Predicting the price of a 16-year old Orion mathematically prints a negative dollar amount.

Influential Observations (Fig 14.13)

An influential observation significantly tilts the entire line of best fit.

In the Orion data, removing car $(2, 169)$ radically changes the slope from $-20.26x$ to $-14.24x$.

Key Fact 14.3: Always graph the scatterplot before calculating formulas!

Predict the Hazard
A real estate agent runs a regression model to predict the selling price of homes ($y$) based on the square footage ($x$). Her model is built using homes ranging from $1,000$ to $3,500$ sq. ft.

A client asks her to use the equation to predict the selling price of a massive $8,000$ sq. ft. mansion.

Should the agent trust this prediction? Justify your stance.

Solution

No. This is a severe case of Extrapolation.

The regression model is entirely ignorant of the housing market dynamics for homes beyond 3,500 sq. ft. Luxury mansions ($8,000$ sq. ft.) often increase in price non-linearly (exponentially). The line of best fit created for middle-class homes will likely grossy under-predict the true value of a mega-mansion. Predictions are only valid within the sampled range of $x$ data.

14.3 Coefficient of Determination ($r^2$)

Total variation in our $y$-variable is mathematically decomposed into the Regression Identity: $SST = SSR + SSE$

SST (Total Sum of Squares) SSR (Regression SS) SSE (Error SS)
$\Sigma(y_i - \overline{y})^2$ $\Sigma(\hat{y}_i - \overline{y})^2$ $\Sigma(y_i - \hat{y}_i)^2$
The total raw variation in the observed values. The variation successfully explained by our regression line. The variation completely unexplained by regression (noise).

The Coefficient of Determination: $r^2 = \frac{SSR}{SST}$

It reports the exact proportion of total variation explained by the regression equation.

Sums of Squares: Computing Protocol

Formula 14.2

  • $SST = \Sigma y_i^2 - \frac{(\Sigma y_i)^2}{n}$
  • $SSR = \frac{[\Sigma x_i y_i - (\Sigma x_i)(\Sigma y_i)/n]^2}{\Sigma x_i^2 - (\Sigma x_i)^2 / n}$
  • $SSE = SST - SSR$

Applied Orion ANOVA

Extracting columns from Table 14.8 directly outputs the identity computationally:

  • $SST = 96,129 - \frac{975^2}{11} = \mathbf{9,708.5}$
  • $SSR = \frac{(-408.9)^2}{20.18} = \mathbf{8,284.4}$
  • $SSE = 9708.5 - 8284.4 = \mathbf{1,424.1}$
$r^2 = 8284.4 \, / \, 9708.5 = 85.3\%$

14.4 The Linear Correlation Coefficient ($r$)

The linear correlation coefficient $r$, also known as the Pearson correlation coefficient, measures the strength and direction of a linear layout.

$$r = \frac{\Sigma x_i y_i - (\Sigma x_i)(\Sigma y_i)/n}{\sqrt{[\Sigma x_i^2 - (\Sigma x_i)^2/n][\Sigma y_i^2 - (\Sigma y_i)^2/n]}}$$

$-1 \le r \le +1$

The absolute value indicates strength, while the sign ($+$ or $-$) dictates direction (matching $b_1$).

Key Fact 14.5: The coefficient of determination equals exactly the square of the linear correlation coefficient ($r^2$).

# --- Input Vectors ---
Age <- c(5,4,6,5,5,5,6,6,2,7,7)
Price <- c(85,103,70,82,89,98,66,95,169,70,48)

# 1. Generate the Linear Model (lm)
model <- lm(Price ~ Age)

# Extract b0 (Intercept) and b1 (Slope) and r-squared directly
summary(model)

# 2. Extract Sums of Squares (SST, SSR, SSE) for ANOVA
anova(model)

# 3. Predict the price of a 4.5 year old car safely (Interpolation)
predict(model, newdata = data.frame(Age = 4.5))

# 4. Calculate Linear Correlation Coefficient (r)
cor(Age, Price)
