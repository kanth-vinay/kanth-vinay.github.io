1) Slope from Two Points

Find the slope \(m\) of the line through the points \(A(3,-2)\) and \(B(9,7)\).
Show Solution

Use \(m=\dfrac{y_2-y_1}{x_2-x_1}=\dfrac{7-(-2)}{9-3}=\dfrac{9}{6}=\boxed{\tfrac{3}{2}}\).

2) Equation from Two Points (slope–intercept form)

Write the equation of the line through \((-4,5)\) and \((2,-1)\) in slope–intercept form \(y=mx+b\).
Show Solution
  1. Slope: \(m=\dfrac{-1-5}{2-(-4)}=\dfrac{-6}{6}=-1\).
  2. Use point \((2,-1)\) in \(y=mx+b\): \(-1=(-1)(2)+b \Rightarrow b=1\).

Equation: \(\boxed{y=-x+1}\).

3) Convert Standard to Slope–Intercept

Convert \(x+4y=12\) to slope–intercept form and state the slope and \(y\)-intercept.
Show Solution

\(4y=12-x \Rightarrow y=-\tfrac14 x+3\).

Slope \(m=\boxed{-\tfrac14}\), intercept \(b=\boxed{3}\).

4) Point–Slope to Prediction

A line has slope \(m=\tfrac{5}{3}\) and passes through \((6,10)\).
(a) Write its equation in slope–intercept form.
(b) Use your model to find \(y\) when \(x=15\).
Show Solution
  1. Point–slope: \(y-10=\tfrac{5}{3}(x-6)\Rightarrow y=\tfrac{5}{3}x-10+10=\tfrac{5}{3}x\).
  2. At \(x=15\): \(y=\tfrac{5}{3}(15)=25\).

Answer: \(\boxed{y=\tfrac{5}{3}x}\) and \(\boxed{25}\).

5) Infant Weight — Line of Best Fit (model from data)

Hannah’s weight (lbs) was recorded at infant visits:
Months0246
Weight (lb)7.29.410.515.9
Build a simple linear model \(y=mx+b\) using the first and last data points.
Show Solution

Using \((0,7.2)\) and \((6,15.9)\):

\(m=\dfrac{15.9-7.2}{6-0}=\dfrac{8.7}{6}=1.45\). Since \(x=0\) gives \(y=7.2\), \(b=7.2\).

Model: \(\boxed{y=1.45x+7.2}\) (lbs, \(x\) in months).

This “two-point” model is a quick best-fit approximation suitable for hand work.

6) Infant Weight — Predict at 12 Months

Using your model from Q5, estimate Hannah’s weight at 12 months.
Show Solution

\(y=1.45(12)+7.2=17.4+7.2=\boxed{24.6\text{ lb (approx.)}}\).

7) Infant Weight — When Will She Reach 45 lb?

Using the same model \(y=1.45x+7.2\), estimate the age \(x\) (months) when \(y=45\) lb.
Show Solution

\(45=1.45x+7.2 \Rightarrow 1.45x=37.8 \Rightarrow x=\dfrac{37.8}{1.45}\approx \boxed{26.1\text{ months}}\).

8) Life Expectancy — Build a Linear Model

Average life expectancy (years) by birth year:
Birth Year1900191019201930194019501960
Life Exp.47.350.054.159.762.968.269.7
(a) Make a linear model using the endpoints. (b) Write it with \(t=\) years since 1900 to keep numbers small.
Show Solution
  1. Using \((1900,47.3)\) and \((1960,69.7)\): slope \(m=\dfrac{69.7-47.3}{1960-1900}=\dfrac{22.4}{60}=0.373\overline{3}\).
  2. Model with year \(x\): \(y=0.373\overline{3}\,x+b\). Plug \(x=1900\): \(47.3=0.373\overline{3}(1900)+b\Rightarrow b\approx -662.03\).
    \(\Rightarrow \boxed{y\approx 0.373\overline{3}\,x-662.03}\).
  3. Using \(t=x-1900\): \(y=0.373\overline{3}\,t+47.3\). This is numerically cleaner.

9) Life Expectancy — Predict for 2025

Using the model from Q8 with \(t=\) years since 1900, estimate life expectancy for someone born in 2025.
Show Solution

Here \(t=2025-1900=125\). So \(y=0.373\overline{3}(125)+47.3\approx 46.67+47.3=\boxed{94.0\text{ years (model extrapolation)}}\).

Linear extrapolation far beyond the data range can overestimate; this is for modeling practice.

10) Study Hours vs Test Grade — Model & Zero-Study Estimate

The table shows study hours and a student’s test grade.
Hours012467
Grade486268829092
(a) Create a linear model using the endpoints. (b) Use it to estimate the grade for a student who studies \(0\) hours.
Show Solution
  1. Endpoints \((0,48)\) and \((7,92)\): \(m=\dfrac{92-48}{7-0}=\dfrac{44}{7}\approx 6.286\).
  2. Intercept \(b=48\) (because \(x=0\) gives the grade). Model: \(\boxed{y\approx 6.286x+48}\).
  3. At \(x=0\): \(y\approx \boxed{48}\) (consistent with the table).
