Quadratic Word Problems

Answer Key

  1. Setup: Let w be the width of the garden. Let l be the length. Given: l = w + 5 and Area = l · w = 300.
    Equation: (w + 5)w = 300
    Solution:

    w2 + 5w = 300
    w2 + 5w - 300 = 0
    Factoring: We need two numbers that multiply to -300 and add to 5. These are 20 and -15.
    (w + 20)(w - 15) = 0
    This gives w = -20 or w = 15.
    Since width cannot be negative, we use w = 15.
    The length is l = w + 5 = 15 + 5 = 20.

    The width is 15 feet and the length is 20 feet.
  2. Setup: The height is h(t) = -16t2 + 144. The ground is at h = 0.
    Equation: 0 = -16t2 + 144
    Solution: (Using Square Root Method)

    16t2 = 144
    t2 = 144 / 16
    t2 = 9
    t = ±√9
    t = 3 or t = -3
    Since time cannot be negative, t = 3.

    It takes 3 seconds for the balloon to hit the ground.
  3. Setup: The height is h(t) = -16t2 + 64t. The maximum height occurs at the vertex.
    Equation: For a parabola at2 + bt + c, the t-coordinate of the vertex is t = -b / (2a).
    Solution:

    Here, a = -16 and b = 64.
    First, find the time to max height:
    t = -64 / (2(-16)) = -64 / -32 = 2 seconds.
    Now, find the height at t=2:
    h(2) = -16(2)2 + 64(2)
    h(2) = -16(4) + 128
    h(2) = -64 + 128 = 64

    The maximum height the ball reaches is 64 feet.
  4. Setup: Let n be the first even integer. Let n + 2 be the next consecutive even integer.
    Equation: n2 + (n + 2)2 = 100
    Solution:

    n2 + (n2 + 4n + 4) = 100
    2n2 + 4n + 4 = 100
    2n2 + 4n - 96 = 0   (Divide by 2)
    n2 + 2n - 48 = 0
    Factor: (n + 8)(n - 6) = 0
    This gives n = -8 or n = 6.
    Case 1: If n = 6, the integers are 6 and 6+2=8.
    Case 2: If n = -8, the integers are -8 and -8+2=-6.

    The two pairs of integers are {6, 8} and {-8, -6}.
  5. Setup: h(t) = -5t2 + 40t + 10. We want to find t when h = 30.
    Equation: 30 = -5t2 + 40t + 10
    Solution:

    Set the equation to 0:
    0 = -5t2 + 40t - 20
    Divide by -5 to simplify:
    0 = t2 - 8t + 4
    Use the Quadratic Formula: a=1, b=-8, c=4.
    t = ( -(-8) ± √((-8)2 - 4(1)(4)) ) / ( 2(1) )
    t = ( 8 ± √(64 - 16) ) / 2 = ( 8 ± √48 ) / 2
    Simplify √48 = √(16·3) = 4√3.
    t = ( 8 ± 4√3 ) / 2 = 4 ± 2√3
    Decimal: t ≈ 4 - 3.46 = 0.54 and t ≈ 4 + 3.46 = 7.46.

    The rocket reaches 30m at t = 4 ± 2√3 seconds (approx. 0.54s and 7.46s).
  6. Setup: P(x) = -2(x - 8)(x - 20). "Break-even" means Profit P = 0.
    Equation: 0 = -2(x - 8)(x - 20)
    Solution:

    Using the Zero Product Property, the profit is 0 when either factor is 0.
    x - 8 = 0   or   x - 20 = 0
    x = 8   or   x = 20

    The company breaks even if they sell 8 T-shirts or 20 T-shirts.
  7. Setup: Let w be the width. Let l be the length. Given: Area = 150 and l = 2w - 5.
    Equation: (2w - 5)w = 150
    Solution:

    2w2 - 5w = 150
    2w2 - 5w - 150 = 0
    Using factoring by grouping (ac = -300, factors -20, 15):
    2w2 - 20w + 15w - 150 = 0
    2w(w - 10) + 15(w - 10) = 0
    (2w + 15)(w - 10) = 0
    w = -15/2 or w = 10.
    Since width cannot be negative, w = 10.
    Length l = 2(10) - 5 = 20 - 5 = 15.

    The width is 10 meters and the length is 15 meters.
  8. Setup: Let b be the distance of the base from the wall. Let h be the height of the ladder on the wall. Given: Hypotenuse c = 17. h = b + 7.
    Equation: Pythagorean Theorem: h2 + b2 = c2 ⇒ (b + 7)2 + b2 = 172
    Solution:

    (b2 + 14b + 49) + b2 = 289
    2b2 + 14b + 49 = 289
    2b2 + 14b - 240 = 0   (Divide by 2)
    b2 + 7b - 120 = 0
    Factor: (b + 15)(b - 8) = 0
    b = -15 or b = 8.
    Since distance cannot be negative, b = 8.

    The base of the ladder is 8 feet from the wall.
  9. Setup: h(t) = -t2 + 8t + 5. We must complete the square to find the vertex.
    Equation: Convert to h(t) = a(t-h)2 + k
    Solution:

    h(t) = (-t2 + 8t) + 5
    h(t) = -1(t2 - 8t) + 5   (Factor out a=-1)
    Half of -8 is -4. (-4)2 = 16.
    h(t) = -1(t2 - 8t + 16 - 16) + 5   (Add and subtract 16 inside)
    h(t) = -1(t2 - 8t + 16) + (-1)(-16) + 5   (Distribute -1 to -16)
    h(t) = -1(t - 4)2 + 16 + 5
    h(t) = -(t - 4)2 + 21
    The vertex is (4, 21). The max height is the k-value.

    The maximum height is 21 (units).
  10. Setup: Let x be the side length of the original square.
    Height of box h = 3.
    Length of box l = x - 3 - 3 = x - 6.
    Width of box w = x - 3 - 3 = x - 6.
    Volume V = l · w · h = 75.
    Equation: (x - 6)(x - 6)(3) = 75
    Solution:

    3(x - 6)2 = 75
    (x - 6)2 = 25   (Divide by 3)
    x - 6 = ±√25   (Take square root)
    x - 6 = 5   or   x - 6 = -5
    x = 11   or   x = 1
    If x=1, the length x-6 would be negative, which is impossible. So, x = 11.

    The side length of the original square was 11 inches.
