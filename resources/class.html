Quiz Question 1

Write an equivalent logarithmic equation for \(10^{-4} = 0.0001\).

  • A) \(\log_{-4}(0.0001) = 10\)
  • B) \(\log_{10}(0.0001) = -4\)
  • C) \(\log_{0.0001}(10) = -4\)
  • D) \(\log_{10}(-4) = 0.0001\)

Solution: Q1

The rule is \(b^y = x \iff \log_b(x) = y\).

Here, the base \(b=10\), the exponent \(y=-4\), and the result \(x=0.0001\).

Plugging these in gives: \(\log_{10}(0.0001) = -4\)

The correct answer is B.

Quiz Question 2

Convert the equation \(T^W = X\) to a logarithmic equation.

  • A) \(\log_W(X) = T\)
  • B) \(\log_X(T) = W\)
  • C) \(\log_T(W) = X\)
  • D) \(\log_T(X) = W\)

Solution: Q2

Using the rule \(b^y = x \iff \log_b(x) = y\).

Here, the base \(b=T\), the exponent \(y=W\), and the result \(x=X\).

This gives the logarithmic form: \(\log_T(X) = W\)

The correct answer is D.

Quiz Question 3

Determine the value of the unknown in \(\log_6(y) = 2\).

  • A) \(y = 12\)
  • B) \(y = 64\)
  • C) \(y = 36\)
  • D) \(y = \sqrt{6}\)

Solution: Q3

Convert back to exponential form using \( \log_b(x) = y \iff b^y=x\).

\(\log_6(y) = 2 \iff 6^2 = y\).

Therefore, \(y = 36\).

The correct answer is C.

Quiz Question 4

Determine the value of the unknown in \(\log_b(7) = -1/3\).

  • A) \(b = -21\)
  • B) \(b = 1/343\)
  • C) \(b = -1/21\)
  • D) \(b = 7^3\)

Solution: Q4

Convert to exponential form: \(b^y=x\).

\(\log_b(7) = -1/3 \iff b^{-1/3} = 7\).

To solve for b, raise both sides to the -3 power:

\((b^{-1/3})^{-3} = 7^{-3}\)

\(b = \frac{1}{7^3} = \frac{1}{343}\)

The correct answer is B.

Quiz Question 7

P=$6500, r=3.95%, t=6 years. What is the future value when compounded continuously?

  • A) $8216.20
  • B) $8239.71
  • C) $8242.02
  • D) $8242.22

Solution: Q7

Continuous Compounding Formula: \(A = Pe^{rt}\)

Plug in the values: P=6500, r=0.0395, t=6

\(A = 6500 \cdot e^{(0.0395 \cdot 6)} = 6500 \cdot e^{0.237} \approx 6500 \cdot 1.26741 \approx \$8242.22\)

The correct answer is D.

Quiz Question 15

Use natural logarithms to solve the equation \(e^{-0.316t} = 8\).

  • A) \(t \approx 7.91\)
  • B) \(t \approx -0.99\)
  • C) \(t \approx 6.58\)
  • D) \(t \approx -6.58\)

Solution: Q15

1. Take the natural log of both sides: \(\ln(e^{-0.316t}) = \ln(8)\)

2. Use the property \(\ln(e^x)=x\): \(-0.316t = \ln(8)\)

3. Solve for t: \(t = \frac{\ln(8)}{-0.316} \approx \frac{2.0794}{-0.316} \approx -6.5805\)

The correct answer is D.

Quiz Question 24

Differentiate the function \(f(x) = x^4 e^{9x}\).

  • A) \(f'(x) = 36x^3 e^{9x}\)
  • B) \(f'(x) = 4x^3 e^{9x} + 9x^4 e^{9x}\)
  • C) \(f'(x) = 4x^3 \cdot 9e^{9x}\)
  • D) \(f'(x) = 4x^3 e^{9x-1}\)

Solution: Q24

Use the Product Rule: \((uv)' = u'v + uv'\). Let \(u=x^4\) and \(v=e^{9x}\).

\(u' = 4x^3\)

\(v' = e^{9x} \cdot 9 = 9e^{9x}\)

\(f'(x) = (4x^3)(e^{9x}) + (x^4)(9e^{9x})\)

Optionally, factor out common terms: \(x^3e^{9x}(4 + 9x)\)

The correct answer is B.

Quiz Question 33

Find an equation of the line tangent to the graph of \(G(x) = e^{-4x}\) at the point (0,1).

  • A) \(y = -4x + 1\)
  • B) \(y = 4x + 1\)
  • C) \(y = x - 4\)
  • D) \(y = e^{-4}x + 1\)

Solution: Q33

1. Find the derivative: \(G'(x) = e^{-4x} \cdot (-4) = -4e^{-4x}\).

2. Find the slope at x=0: \(m = G'(0) = -4e^{0} = -4\).

3. Use point-slope form: \(y - y_1 = m(x - x_1)\)

\(y - 1 = -4(x - 0)\)
Equation: \(y = -4x + 1\)

The correct answer is A.
