1. Basic Sine and Cosine (Amplitude & Vertical Shift)

6 introductory problems focused on vertical transformations.

Problem 1

What is the amplitude of the function \(y = -3 \sin(x) + 4\)?

Solution / Reasoning: Amplitude is defined as half the distance between the maximum and minimum values, which corresponds to the absolute value of the leading coefficient \(|a|\). Thus, \(|-3| = 3\). Amplitude is always positive.
Problem 2

Identify the equation of the midline for the function \(y = 5 \cos(x) - 2\).

Solution / Reasoning: The midline is the horizontal center of the graph, determined by the vertical shift \(k\). Here, the graph is shifted down 2 units, making the midline \(y = -2\).
Problem 3

What is the range of the function \(y = 2 \sin(x) + 1\)?

Solution / Reasoning: The basic sine function has a range of \([-1, 1]\). Multiplying by 2 stretches it to \([-2, 2]\). Adding 1 shifts the entire range up by 1, resulting in \([-1, 3]\).
Problem 4

Which of the following functions has a maximum value of \(7\) and a minimum value of \(-1\)?

Solution / Reasoning: The midline is the average of the max and min: \((7 + (-1))/2 = 3\). The amplitude is half the distance between max and min: \((7 - (-1))/2 = 4\). The function must have \(a=4\) and \(k=3\). Option C matches.
Problem 5

Identify the equation for a graph that starts at a maximum when \(x = 0\), has an amplitude of \(2\), and a midline of \(y = 0\).

Solution / Reasoning: The parent function that starts at a maximum at \(x=0\) is the cosine function, \(\cos(x)\). An amplitude of 2 implies the coefficient is 2. Thus, \(y = 2 \cos(x)\).
Problem 6

If the graph of \(y = \sin(x)\) is reflected over the x-axis and shifted up 5 units, what is its new equation?

Solution / Reasoning: Reflecting over the x-axis multiplies the output by \(-1\) (yielding \(-\sin(x)\)). Shifting up 5 adds 5 to the outside of the function, giving \(y = -\sin(x) + 5\).

2. Period and Phase Shifts (Sine and Cosine)

6 intermediate problems tackling horizontal transformations.

Problem 7

What is the period of the function \(y = \sin(4x)\)?

Solution / Reasoning: The period of a sine or cosine function is given by \(2\pi/|b|\). Here, \(b = 4\), so the period is \(2\pi/4 = \pi/2\).
Problem 8

What is the phase shift of the function \(y = \cos(2(x - \pi/4))\)?

Solution / Reasoning: When the equation is factored as \(y = a \cos(b(x - h)) + k\), the phase shift is \(h\). Here, \(h = \pi/4\), meaning a shift to the right by \(\pi/4\).
Problem 9

Find the period of the function \(y = -3 \cos(\pi x)\).

Solution / Reasoning: The period is \(2\pi / |b|\). With \(b = \pi\), the period is \(2\pi / \pi = 2\). The period is a rational number, not involving \(\pi\).
Problem 10

Which equation represents a sine wave with an amplitude of \(1\) and a period of \(4\pi\)?

Solution / Reasoning: If the period is \(4\pi\), we set \(2\pi / b = 4\pi\). Solving for \(b\) gives \(b = 1/2\). Therefore, the equation is \(y = \sin(x/2)\).
Problem 11

Where does the graph of \(y = \sin(x - \pi/2)\) have its first positive x-intercept?

Solution / Reasoning: The parent function \(y = \sin(x)\) has x-intercepts at \(0, \pi, 2\pi\). Shifting right by \(\pi/2\) moves these intercepts to \(\pi/2, 3\pi/2, 5\pi/2\). The first positive intercept is at \(x = \pi/2\).
Problem 12

What is the phase shift of \(y = 2 \sin(3x + \pi)\)?

Solution / Reasoning: First, factor out the \(3\) from the input: \(y = 2 \sin(3(x + \pi/3))\). The \(+ \pi/3\) inside the parentheses indicates a horizontal shift to the left by \(\pi/3\).

3. Tangent and Cotangent Graphs

6 problems focusing on asymptotes, periods, and behavior of tan/cot.

Problem 13

What is the period of the function \(y = \tan(x/2)\)?

Solution / Reasoning: The parent tangent function has a period of \(\pi\). The period formula for tangent is \(\pi/|b|\). Here, \(b = 1/2\), so the period is \(\pi / (1/2) = 2\pi\).
Problem 14

Identify the vertical asymptotes of \(y = \tan(x)\) on the interval \([0, 2\pi]\).

Solution / Reasoning: Tangent is defined as \(\sin(x)/\cos(x)\). Vertical asymptotes occur where \(\cos(x) = 0\). On the interval \([0, 2\pi]\), cosine is zero at \(x = \pi/2\) and \(x = 3\pi/2\).
Problem 15

Where does the graph of \(y = \cot(x)\) have an x-intercept?

Solution / Reasoning: Cotangent is \(\cos(x)/\sin(x)\). It crosses the x-axis (equals zero) wherever \(\cos(x) = 0\) and \(\sin(x) \neq 0\). The cosine function is zero at \(x = \pi/2\).
Problem 16

What is the domain of \(y = \tan(2x)\)? (where \(k\) is an integer)

Solution / Reasoning: Asymptotes of \(\tan(\theta)\) occur at \(\theta = \pi/2 + k\pi\). Set \(2x = \pi/2 + k\pi\) and solve for \(x\): dividing by 2 gives \(x = \pi/4 + k\pi/2\). The domain is all real numbers except these values.
Problem 17

Find the period of the function \(y = -4 \cot(\pi x)\).

Solution / Reasoning: The period of cotangent is \(\pi/|b|\). Since \(b = \pi\), the period is \(\pi/\pi = 1\).
Problem 18

Which parent function has vertical asymptotes at \(x = k\pi\) (where \(k\) is an integer)?

Solution / Reasoning: Vertical asymptotes occur where the denominator is zero. For \(\cot(x) = \cos(x)/\sin(x)\), this happens when \(\sin(x) = 0\), which is exactly at \(x = 0, \pi, 2\pi\dots\) or \(x = k\pi\).

4. Synthesis & Writing Equations

6 challenging problems connecting graphs to equations and real-world models.

Problem 19

Write a cosine function with an amplitude of \(5\), period of \(2\pi/3\), and a midline of \(y = 1\). (Assume no phase shift).

Solution / Reasoning: Amplitude \(a=5\), midline \(k=1\). To find \(b\), use \(2\pi/b = \text{period}\). So, \(2\pi/b = 2\pi/3\), meaning \(b=3\). The function is \(y = 5 \cos(3x) + 1\).
Problem 20

A Ferris wheel has a radius of \(20\)m and its center is \(25\)m above the ground. It completes one revolution every \(2\) minutes. If a rider starts at the very bottom, which function models their height \(h(t)\) over time \(t\) in minutes?

Solution / Reasoning: The radius is the amplitude (\(20\)), the center is the midline (\(y=25\)). Starting at the bottom means starting at a minimum, so we use a negative cosine function. Period = \(2\) mins, so \(b = 2\pi/2 = \pi\). Thus, \(h(t) = -20 \cos(\pi t) + 25\).
Problem 21

The graph of \(y = a \sin(b(x-h)) + k\) has a maximum at \((0, 5)\) and an adjacent minimum at \((\pi, -1)\). Find the values of \(k\) and \(a\) (assuming \(a > 0\)).

Solution / Reasoning: Midline \(k\) is the average of max and min: \((5 + (-1))/2 = 4/2 = 2\). Amplitude \(a\) is half the distance: \((5 - (-1))/2 = 6/2 = 3\). So \(k=2\), \(a=3\).
Problem 22

A trigonometric graph reaches a maximum at \(x = 2\) and its next adjacent minimum at \(x = 5\). What is the period of the function?

Solution / Reasoning: The horizontal distance from a maximum to the next minimum is exactly half of one period. Distance = \(5 - 2 = 3\). Thus, the full period is \(3 \times 2 = 6\).
Problem 23

The graph of \(y = \sin(x + \pi/2)\) is identical to the graph of which parent function?

Solution / Reasoning: A horizontal shift of the sine wave left by \(\pi/2\) perfectly aligns its peaks with the y-axis, making it equivalent to the cosine function. This is a core co-function identity: \(\sin(x + \pi/2) = \cos(x)\).
Problem 24

Ocean tides reach a high of \(10\) ft at \(t = 0\) hours and a low of \(2\) ft at \(t = 6\) hours. Which equation models the water depth \(y\) at time \(t\)?

Solution / Reasoning: Max is 10, Min is 2. Midline is \((10+2)/2 = 6\). Amplitude is \((10-2)/2 = 4\). Starts at a maximum at \(t=0\), meaning a positive cosine graph. Half period is 6 hrs (from max to min), so full period is 12 hrs. \(b = 2\pi/12 = \pi/6\). Thus, \(y = 4 \cos(\frac{\pi}{6} t) + 6\).

5. Algebra II Free Response (AP Style formatting)

6 multi-step FRQs designed to test rigorous synthesis of graphical concepts.

Problem 25

FRQ 1. A mass on a spring oscillates up and down. Its position is modeled by \(y(t) = -4 \cos(\pi t) + 5\). State the amplitude, period, and midline. Find the first time \(t > 0\) the mass reaches its highest point.

Sample Solution Outline:
Amplitude: \(|-4| = 4\).
Period: \(2\pi/\pi = 2\).
Midline: \(y = 5\).
Highest point occurs when \(y\) is at its maximum (\(5+4=9\)). Set \(-4 \cos(\pi t) + 5 = 9 \implies -4\cos(\pi t) = 4 \implies \cos(\pi t) = -1\). The first positive time cosine equals -1 is when the angle is \(\pi\). Thus, \(\pi t = \pi \implies t = 1\) second.
Problem 26

FRQ 2. Given \(f(x) = 3 \tan(x/2)\), find the period and write the equations of the first two positive vertical asymptotes. Explain mathematically why the tangent function has asymptotes based on the sine and cosine functions.

Sample Solution Outline:
Period: \(\pi / (1/2) = 2\pi\).
Asymptotes occur when the argument of tan equals \(\pi/2, 3\pi/2\), etc. So \(x/2 = \pi/2 \implies x = \pi\). Next one: \(x/2 = 3\pi/2 \implies x = 3\pi\). First two positive asymptotes are \(x=\pi\) and \(x=3\pi\).
Explanation: Tangent is the ratio \(\sin(x)/\cos(x)\). When \(\cos(x) = 0\), the denominator is zero, causing the function's output to approach infinity (an asymptote).
Problem 27

FRQ 3. The average monthly temperature in a city varies sinusoidally. The maximum temperature is \(80^\circ\)F in July (\(t = 7\)), and the minimum is \(30^\circ\)F in January (\(t = 1\)). Write a cosine function \(T(t)\) modeling this temperature, where \(t\) is months (Jan=1).

Sample Solution Outline:
Midline: \((80+30)/2 = 55\).
Amplitude: \((80-30)/2 = 25\).
Period: A full year is 12 months, so \(b = 2\pi/12 = \pi/6\).
Phase Shift: Max occurs at \(t=7\), so we can use a positive cosine shifted right by 7.
Equation: \(T(t) = 25 \cos(\frac{\pi}{6}(t - 7)) + 55\). (Alternatively, starting at min at \(t=1\) yields \(T(t) = -25 \cos(\frac{\pi}{6}(t - 1)) + 55\)).
Problem 28

FRQ 4. Describe the specific sequence of transformations required to map the parent function \(y = \sin(x)\) to the function \(y = -2 \sin(3(x - \pi/6)) + 1\). Then, state the range of the new function.

Sample Solution Outline:
Transformations (order matters for horizontal/vertical independently):
1. Horizontal shrink by a factor of \(1/3\).
2. Horizontal shift right by \(\pi/6\).
3. Vertical stretch by a factor of \(2\).
4. Reflection over the x-axis.
5. Vertical shift up by \(1\).
Range: Amplitude is 2, Midline is 1. Max = \(1+2=3\), Min = \(1-2=-1\). Range is \([-1, 3]\).
Problem 29

FRQ 5. Let \(g(x) = \cot(2x)\). Determine the period, all x-intercepts on the interval \([0, \pi]\), and all vertical asymptotes on the interval \([0, \pi]\).

Sample Solution Outline:
Period: \(\pi/2\).
Asymptotes: Parent \(\cot(x)\) has asymptotes at \(k\pi\). Set \(2x = k\pi \implies x = k\pi/2\). On \([0, \pi]\), asymptotes are \(x = 0, \pi/2, \pi\).
x-intercepts: Parent \(\cot(x)\) has intercepts at \(\pi/2 + k\pi\). Set \(2x = \pi/2 + k\pi \implies x = \pi/4 + k\pi/2\). On \([0, \pi]\), the intercepts are \(x = \pi/4\) and \(x = 3\pi/4\).
Problem 30

FRQ 6. Consider the system of equations given by \(f(x) = 2 \sin(x)\) and \(g(x) = 1\). Find all points of intersection on the interval \([0, 2\pi]\).

Sample Solution Outline:
Set the equations equal: \(2 \sin(x) = 1 \implies \sin(x) = 1/2\).
We look for angles on the unit circle between \(0\) and \(2\pi\) where the y-coordinate (sine) is positive \(1/2\).
This occurs in Quadrants I and II. The reference angle is \(\pi/6\).
The solutions are \(x = \pi/6\) and \(x = \pi - \pi/6 = 5\pi/6\).
Intersection points: \((\pi/6, 1)\) and \((5\pi/6, 1)\).
