Homework Problem: Q1 - Terminology

Fill in the blank: The value of a statistic used to estimate a parameter is called a(n) _________ of the parameter.
  • biased estimator
  • unbiased estimator
  • point estimate
  • confidence-interval estimate

Click to Reveal Solution

Answer: point estimate

Explanation: A point estimate (like the sample mean, \(\bar{x}\)) is our single best numerical guess for the value of the population parameter (like the population mean, \(\mu\)).

Homework Problem: Q2 - Margin of Error

When estimating an unknown parameter, what does the margin of error indicate?
  • A. It indicates the precision of an estimate for the value of an unknown statistic.
  • B. It indicates the level of confidence of an estimate for the value of the unknown parameter.
  • C. It indicates the accuracy of an estimate for the value of the unknown parameter.
  • D. It indicates the value of the statistic used to estimate the parameter.

Click to Reveal Solution

Answer: C

Explanation: The margin of error quantifies the accuracy of our estimate. It gives us a sense of how close our point estimate is likely to be to the true population parameter. A smaller margin of error implies a more precise (and thus more accurate) estimate.

Homework Problem: Q3 - Confidence Level

Suppose that you take 1000 simple random samples from a population and that, for each sample, you obtain a 95% confidence interval for an unknown parameter. Approximately how many of those confidence intervals will contain the value of the unknown parameter?

Click to Reveal Solution

Answer: Approximately 950

Explanation: A 95% confidence level means that the *method* we use to construct the interval will successfully capture the true population parameter 95% of the time. Therefore, if we repeat the process 1000 times, we expect about 95% of those intervals, or \(1000 \times 0.95 = 950\), to contain the true parameter.

Homework Problem: Q6 - z-Interval (σ Known)

Given \(\bar{x}=30\), \(n=100\), and \(\sigma=2\), find a 95% confidence interval and the margin of error.

Click to Reveal Solution

Manual Calculation

Formula: \(\bar{x} \pm z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}\)

  1. Critical Value: For 95% CI, \(z_{0.025} = 1.96\).
  2. Margin of Error (E):
    \( E = 1.96 \cdot \frac{2}{\sqrt{100}} = \mathbf{0.392} \)
  3. Interval: \( 30 \pm 0.392 \Rightarrow \mathbf{(29.608, 30.392)} \)

RStudio Commands

x_bar<-30; n<-100; sigma<-2
z_crit <- qnorm(0.975)
E <- z_crit * sigma / sqrt(n)
E # Margin of Error
c(x_bar - E, x_bar + E) # CI

Homework Problem: Q7 - z-Interval (Normal Pop.)

Given \(\bar{x}=37\), \(n=25\), \(\sigma=4\), and x is normally distributed. Find a 95% CI.

Click to Reveal Solution

Manual Calculation

Formula: \(\bar{x} \pm z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}\)

  1. Critical Value: For 95% CI, \(z_{0.025} = 1.96\).
  2. Margin of Error (E):
    \( E = 1.96 \cdot \frac{4}{\sqrt{25}} = \mathbf{1.568} \)
  3. Interval: \( 37 \pm 1.568 \Rightarrow \mathbf{(35.432, 38.568)} \)

RStudio Commands

x_bar<-37; n<-25; sigma<-4
z_crit <- qnorm(0.975)
E <- z_crit * sigma / sqrt(n)
c(x_bar - E, x_bar + E)

Homework Problem: Q8 & Q9 - Interval Concepts

Q8: Find the confidence level and \(\alpha\) for a 99% confidence interval.

Q9: Which confidence level, 95% or 99%, gives a more accurate (more precise) estimate of \(\mu\)?

Click to Reveal Answers

Answers

Q8: For a 99% confidence interval, the confidence level is 0.99. Alpha (\(\alpha\)) is the significance level, which is \(1 - \text{confidence level}\). So, \(\alpha = 1 - 0.99 = \mathbf{0.01}\).

Q9: The 95% confidence level gives a more accurate (i.e., more precise or narrower) estimate. To be more confident (99%), we must create a wider interval to have a better chance of capturing \(\mu\). This wider interval is less precise.

Homework Problem: Q10 - True or False

a. The length of a CI can be determined if you only know the margin of error.
b. The margin of error can be determined if you only know the length of the CI.
c. The CI can be obtained if you only know the margin of error.
d. The CI can be obtained if you know the margin of error and the sample mean.

Click to Reveal Answers

Answers

a. True. Length = 2 × (Margin of Error).

b. True. Margin of Error = Length / 2.

c. False. You need the center of the interval, which is the sample mean (\(\bar{x}\)).

d. True. The interval is \(\bar{x} \pm \text{Margin of Error}\).

Homework Problem: Q13 - Sample Size

For a 95% CI, what sample size is needed for a margin of error of 4, if we estimate \(\sigma \approx s = 12.64\)?

Click to Reveal Solution

Manual Calculation

Formula: \( n = \left( \frac{z_{\alpha/2} \cdot \sigma}{E} \right)^2 \)

  1. Values: E=4, \(\sigma \approx 12.64\), \(z_{0.025} = 1.96\).
  2. Calculate: \( n = \left( \frac{1.96 \cdot 12.64}{4} \right)^2 \approx 38.36 \)
  3. Round Up: Always round up for sample size.
    Required \(n = \mathbf{39}\).

RStudio Commands

E <- 4; s <- 12.64
z_crit <- qnorm(0.975)
n_req <- (z_crit * s / E)^2
ceiling(n_req) # Rounds up

Homework Problem: Q14 & Q17 - Final Concepts

Q14: Which confidence level leads to the largest CI width? (90%, 95%, 98%, 99%)

Q17: What is the formula for the margin of error when \(\sigma\) is unknown?

Click to Reveal Answers

Answers

Q14: The 99% confidence level. To be more confident, you need a wider interval to increase the chance of capturing the true parameter. A wider interval corresponds to a larger margin of error.

Q17: When \(\sigma\) is unknown, we use the sample standard deviation (\(s\)) and a t-critical value. The formula for the margin of error is:
\[ E = t_{\alpha/2} \frac{s}{\sqrt{n}} \]

Solutions using RStudio

Q15: Find \(t_{0.005}\) with df=16 (Area to the RIGHT is 0.005)

# R needs area to the LEFT (1 - 0.005 = 0.995)
qt(0.995, df = 16) # Result: ~2.921

Q16: Find t-value with area 0.10 to right, df=46

# Area to the LEFT is 1 - 0.10 = 0.90
qt(0.90, df = 46) # Result: ~1.300

Q18: Find 98% CI for \(\bar{x}=34, n=36, s=5\)

df <- 35 # n-1
# For 98% CI, left tail is 0.01, so we need qt(0.99)
t_crit <- qt(0.99, df=df)
E <- t_crit * 5 / sqrt(36)
c(34 - E, 34 + E) # Result: (31.969, 36.031)
