Directions

This is an original ACT-style Math practice test. It contains 45 multiple-choice questions designed for a 50-minute sitting. ACT Math allows calculator use, but every problem is written to be solvable without advanced calculator features.

Choose the best answer for each question. Responses save locally in this browser. Final submission locks all responses and displays diagnostic feedback by skill category.

Practice scoring counts all 45 questions. Official ACT forms may include unscored field-test questions, so this raw score and practice index are diagnostic only.
