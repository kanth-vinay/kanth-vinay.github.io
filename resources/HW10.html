Problem 1

Find (a) the general solution and (b) the particular solution for the given initial condition.
\(y^{\prime}=7x^{6};\quad y(0)=6\)

Solution:

(a) General Solution

The problem gives us the derivative \(y^{\prime}\) (which is the same as \(\frac{dy}{dx}\)) and asks for the original function \(y\). To find \(y\), we need to find the antiderivative, or integrate, \(y^{\prime}\) with respect to \(x\).

\[y = \int y^{\prime} dx = \int 7x^{6} dx\]

Using the power rule for integration, \(\int x^n dx = \frac{x^{n+1}}{n+1} + C\), where \(n=6\):

\[y = 7 \left( \frac{x^{6+1}}{6+1} \right) + C = 7 \left( \frac{x^7}{7} \right) + C\]

The 7s cancel out, leaving:

\[y = x^7 + C\]

This is the general solution, because \(C\) can be any constant.

(b) Particular Solution

To find the particular solution, we use the initial condition \(y(0)=6\). This means "when \(x=0\), \(y=6\)". We plug these values into our general solution to solve for \(C\).

\(y = x^7 + C\)

\(6 = (0)^7 + C\)

\(6 = 0 + C\)

\(C = 6\)

Now we substitute this specific value of \(C\) back into the general solution.

The particular solution is \(y = x^7 + 6\).

Problem 2

Find (a) the general solution and (b) the particular solution for the given initial condition.
\(y^{\prime}=\frac{5}{x}+2x^{4}-2x^{6},\quad y(1)=-9\)

Solution:

(a) General Solution

First, we integrate \(y^{\prime}\) to find \(y\). We can integrate each term separately.

\[y = \int \left( \frac{5}{x}+2x^{4}-2x^{6} \right) dx\]

\[y = \int \frac{5}{x} dx + \int 2x^{4} dx - \int 2x^{6} dx\]

Remember these integration rules:

  • \(\int \frac{k}{x} dx = k \ln|x| + C\)
  • \(\int kx^n dx = k \frac{x^{n+1}}{n+1} + C\)

Applying these rules:

\[y = 5 \ln|x| + 2 \left( \frac{x^{4+1}}{4+1} \right) - 2 \left( \frac{x^{6+1}}{6+1} \right) + C\]

\[y = 5 \ln|x| + 2 \left( \frac{x^5}{5} \right) - 2 \left( \frac{x^7}{7} \right) + C\]

The general solution is \(y = 5 \ln|x| + \frac{2}{5}x^5 - \frac{2}{7}x^7 + C\).

(b) Particular Solution

We use the initial condition \(y(1)=-9\). Plug \(x=1\) and \(y=-9\) into the general solution.

\(-9 = 5 \ln|1| + \frac{2}{5}(1)^5 - \frac{2}{7}(1)^7 + C\)

We know that \(\ln(1) = 0\), so:

\(-9 = 5(0) + \frac{2}{5}(1) - \frac{2}{7}(1) + C\)

\(-9 = 0 + \frac{2}{5} - \frac{2}{7} + C\)

Now, we solve for \(C\). We need a common denominator for the fractions (which is 35).

\(-9 = \frac{2 \cdot 7}{5 \cdot 7} - \frac{2 \cdot 5}{7 \cdot 5} + C\)

\(-9 = \frac{14}{35} - \frac{10}{35} + C\)

\(-9 = \frac{4}{35} + C\)

\(C = -9 - \frac{4}{35}\)

To subtract, we put 9 over 35:

\[C = -\frac{9 \cdot 35}{35} - \frac{4}{35} = -\frac{315}{35} - \frac{4}{35} = -\frac{319}{35}\]

Now, substitute this \(C\) back into the general solution.

The particular solution is \(y = 5 \ln|x| + \frac{2}{5}x^5 - \frac{2}{7}x^7 - \frac{319}{35}\).

Problem 3

Show that \(y=- \ln x+4x-3\) is a solution of \(y^{\prime\prime}-\frac{1}{x^{2}}=0\).

Solution:

To show this, we need to find the first derivative (\(y^{\prime}\)) and the second derivative (\(y^{\prime\prime}\)) of the given function \(y\). Then we plug \(y^{\prime\prime}\) into the differential equation and see if it holds true.

Step 1: Find \(y^{\prime}\)

Given: \(y = - \ln x + 4x - 3\)

We differentiate term by term. Remember \(\frac{d}{dx}(\ln x) = \frac{1}{x}\) and \(\frac{d}{dx}(kx) = k\).

\[y^{\prime} = \frac{d}{dx}(- \ln x + 4x - 3) = -\frac{1}{x} + 4 - 0\]

\(y^{\prime} = -\frac{1}{x} + 4\)

Step 2: Find \(y^{\prime\prime}\)

Now we differentiate \(y^{\prime}\). It's easier if we write \(y^{\prime}\) using a negative exponent: \(y^{\prime} = -x^{-1} + 4\).

Using the power rule \(\frac{d}{dx}(x^n) = nx^{n-1}\):

\[y^{\prime\prime} = \frac{d}{dx}(-x^{-1} + 4) = -(-1)x^{-1-1} + 0\]

\[y^{\prime\prime} = 1x^{-2} = \frac{1}{x^2}\]

Step 3: Substitute into the differential equation

The differential equation is \(y^{\prime\prime}-\frac{1}{x^{2}}=0\).

We substitute our result for \(y^{\prime\prime}\):

\[\left( \frac{1}{x^2} \right) - \frac{1}{x^2} = 0\]

\[0 = 0\]

Since this is a true statement, we have shown that \(y = - \ln x + 4x - 3\) is indeed a solution to the differential equation.

(Note: This solution is valid for \(x > 0\), as required by the \(\ln x\) function.)

4b. a) Find the general solution of the differential equation \(\frac{dN}{dt}=0.35N.\)
b) Check the solution by substituting into the differential equation.

Solution (4b):

(a) General Solution

This is a separable differential equation. We want to get all the \(N\) terms on one side (with \(dN\)) and all the \(t\) terms on the other (with \(dt\)).

\[\frac{dN}{dt}=0.35N\]

Divide by \(N\) and multiply by \(dt\):

\[\frac{1}{N} dN = 0.35 dt\]

Now, integrate both sides:

\[\int \frac{1}{N} dN = \int 0.35 dt\]

\[\ln|N| = 0.35t + C_1\] (where \(C_1\) is the constant of integration)

To solve for \(N\), we make each side the exponent of \(e\):

\[e^{\ln|N|} = e^{0.35t + C_1}\]

\[|N| = e^{0.35t} \cdot e^{C_1}\]

Since \(e^{C_1}\) is just some positive constant, we can rename it. Let \(C = \pm e^{C_1}\). This new constant \(C\) can be any non-zero constant. (In growth models, we usually just let \(C\) represent the initial value, which can be positive).

The general solution is \(N(t) = Ce^{0.35t}\).

(This is the standard form for exponential growth, \(\frac{dy}{dt} = ky \rightarrow y = Ce^{kt}\). Here, \(k=0.35\).)

(b) Check the solution

Our solution is \(N = Ce^{0.35t}\).

The differential equation is \(\frac{dN}{dt} = 0.35N\).

First, let's find the derivative of our solution, \(\frac{dN}{dt}\):

\[\frac{dN}{dt} = \frac{d}{dt}(Ce^{0.35t}) = C \cdot (e^{0.35t} \cdot 0.35)\] (using the chain rule)

\[\frac{dN}{dt} = 0.35Ce^{0.35t}\]

Now, let's substitute both \(N\) and \(\frac{dN}{dt}\) into the original equation:

\[\frac{dN}{dt} = 0.35N\]

\[ (0.35Ce^{0.35t}) = 0.35(Ce^{0.35t}) \]

Both sides are identical. The solution checks.

Problem 5

\[s(t) = \int \frac{ds}{dt} dt = \int (11t^2 + 3t - 7) dt\]

Using the power rule for integration:

\[s(t) = 11 \left( \frac{t^3}{3} \right) + 3 \left( \frac{t^2}{2} \right) - 7t + C\]

The general solution is \(s(t) = \frac{11}{3}t^3 + \frac{3}{2}t^2 - 7t + C\).

\(103 = \frac{11}{3}(0)^3 + \frac{3}{2}(0)^2 - 7(0) + C\)

\(103 = 0 + 0 - 0 + C\)

\(C = 103\)

Substitute this \(C\) back into the general solution.

The particular solution is \(s(t) = \frac{11}{3}t^3 + \frac{3}{2}t^2 - 7t + 103\).

Problem 8

Solve by separating variables.
\(\frac{dy}{dx}=3x^{2}y\)

Solution:

Step 1: Separate the variables.

We need to get all \(y\) terms with \(dy\) and all \(x\) terms with \(dx\).

Divide both sides by \(y\):

\[\frac{1}{y} \frac{dy}{dx} = 3x^2\]

Multiply both sides by \(dx\):

\[\frac{1}{y} dy = 3x^2 dx\]

Step 2: Integrate both sides.

\[\int \frac{1}{y} dy = \int 3x^2 dx\]

\[\ln|y| = 3 \left( \frac{x^3}{3} \right) + C_1\]

\[\ln|y| = x^3 + C_1\]

Step 3: Solve for y.

Make both sides the exponent of \(e\):

\[e^{\ln|y|} = e^{x^3 + C_1}\]

\[|y| = e^{x^3} \cdot e^{C_1}\]

Let \(C = \pm e^{C_1}\). This is our new constant.

The solution is \(y = Ce^{x^3}\).

Problem 9

Solve by separating variables.
\(5y^{4}\frac{dy}{dx}=2x\)

Solution:

Step 1: Separate the variables.

The \(y\) terms are already on the left and the \(x\) term is on the right. We just need to multiply by \(dx\).

\[5y^4 dy = 2x dx\]

Step 2: Integrate both sides.

\[\int 5y^4 dy = \int 2x dx\]

Using the power rule:

\[5 \left( \frac{y^5}{5} \right) = 2 \left( \frac{x^2}{2} \right) + C\]

\[y^5 = x^2 + C\]

Step 3: Solve for y.

Take the 5th root of both sides:

\[y = \sqrt[5]{x^2 + C}\]

This can also be written as \(y = (x^2 + C)^{1/5}\).

Problem 10

Solve by separating variables.
\(\frac{dy}{dx}=\frac{6x}{y}\)

Solution:

Step 1: Separate the variables.

Multiply by \(y\) and multiply by \(dx\).

\[y dy = 6x dx\]

Step 2: Integrate both sides.

\[\int y dy = \int 6x dx\]

\[\frac{y^2}{2} = 6 \left( \frac{x^2}{2} \right) + C_1\]

\[\frac{y^2}{2} = 3x^2 + C_1\]

Step 3: Solve for y.

Multiply the entire equation by 2 to clear the fraction:

\[y^2 = 2(3x^2 + C_1) = 6x^2 + 2C_1\]

Since \(C_1\) is an arbitrary constant, \(2C_1\) is also just an arbitrary constant. Let's call it \(C\).

\[y^2 = 6x^2 + C\]

Now, take the square root of both sides. Remember to include both the positive and negative roots.

\[y = \pm \sqrt{6x^2 + C}\]

So, the two solutions are \(y = \sqrt{6x^2 + C}\) and \(y = -\sqrt{6x^2 + C}\).

\(C = 9\)

Problem 13

Debra deposits \$1700 into an account that earns interest at a rate of 3.83% compounded continuously.
a) Write the differential equation that represents \(A(t)\) the value of Debra's account after t years.
b) Find the particular solution of the differential equation from part (a).
c) Find \(A(5)\) and \(A^{\prime}(5)\)
d) Find \(\frac{A^{\prime}(5)}{A(5)},\) and explain what this number represents.

Solution:

a) The differential equation

"Compounded continuously" means the rate of change of the account value (\(\frac{dA}{dt}\)) is proportional to the current account value (\(A\)). This is the exponential growth model \(\frac{dA}{dt} = kA\).

The interest rate is \(k = 3.83\% = 0.0383\).

The differential equation is \(\frac{dA}{dt} = 0.0383A\).

b) The particular solution

The general solution is \(A(t) = Ce^{kt} = Ce^{0.0383t}\).

The initial condition is the deposit, \(A(0) = 1700\).

As we saw in problems 6 and 7, the constant \(C\) is just the initial value.

The particular solution is \(A(t) = 1700e^{0.0383t}\).

c) Find \(A(5)\) and \(A^{\prime}(5)\)

To find \(A(5)\), we plug \(t=5\) into our solution \(A(t)\):

\[A(5) = 1700e^{0.0383(5)} = 1700e^{0.1915}\]

\(A(5) \approx 1700(1.21106) \approx 2058.805\)

Rounding to two decimal places, \(A(5) = \$2058.81\).


To find \(A^{\prime}(5)\), we first need the derivative function \(A^{\prime}(t)\). We can get this from the original differential equation: \(A^{\prime}(t) = \frac{dA}{dt} = 0.0383A\).

So, \(A^{\prime}(5) = 0.0383 \cdot A(5)\).

\(A^{\prime}(5) = 0.0383 \cdot (2058.805...) \approx 78.852\)

Rounding to two decimal places, \(A^{\prime}(5) = \$78.85\) per year.

(This means that after 5 years, the account is growing at a rate of $78.85 per year.)

d) Find \(\frac{A^{\prime}(5)}{A(5)}\) and explain

We just calculated these values, but we don't even need to. Look at the original differential equation:

\[\frac{dA}{dt} = 0.0383A\]

If we divide both sides by \(A\), we get:

\[\frac{1}{A} \frac{dA}{dt} = 0.0383\]

Or \(\frac{A^{\prime}(t)}{A(t)} = 0.0383\).

This is true for any time \(t\), including \(t=5\).

\[\frac{A^{\prime}(5)}{A(5)} = 0.0383\]

Explanation: This number, 0.0383, is the value \(k\). It represents the continuous growth rate (or the interest rate). It's the "relative rate of change" â€” the rate of growth as a percentage of the current amount.

Problem 14

A firm's marginal profit, P, as a function of its total cost, C, is given by \(\frac{dP}{dC}=\frac{-224}{{(C+1)}^{3/2}}\)
a) Find the profit function, \(P(C)\) if \(P=\$8\) when \(C=\$48\)
b) At what cost will the firm break even \((P=0)\)?

Solution:

a) Find the profit function \(P(C)\)

We are given the derivative (marginal profit) and need to find the profit function \(P\) by integrating with respect to \(C\).

First, rewrite the function using a negative exponent:

\[\frac{dP}{dC} = -224(C+1)^{-3/2}\]

\[P(C) = \int -224(C+1)^{-3/2} dC\]

This requires a u-substitution. Let \(u = C+1\). Then \(du = dC\).

\[P(C) = \int -224 u^{-3/2} du\]

Using the power rule for integration:

\[P(C) = -224 \left( \frac{u^{-3/2 + 1}}{-3/2 + 1} \right) + K = -224 \left( \frac{u^{-1/2}}{-1/2} \right) + K\]

Dividing by -1/2 is the same as multiplying by -2:

\[P(C) = -224 \cdot (-2) u^{-1/2} + K = 448u^{-1/2} + K\]

Now, substitute back \(u = C+1\):

\[P(C) = 448(C+1)^{-1/2} + K = \frac{448}{\sqrt{C+1}} + K\]

This is the general solution. Now find the particular solution using \(P=8\) when \(C=48\).

\[8 = \frac{448}{\sqrt{48+1}} + K\]

\[8 = \frac{448}{\sqrt{49}} + K\]

\[8 = \frac{448}{7} + K\]

\(8 = 64 + K\)

\(K = 8 - 64 = -56\)

The profit function is \(P(C) = \frac{448}{\sqrt{C+1}} - 56\).

b) Find the break-even cost

Break-even means profit \(P=0\). We set our profit function to 0 and solve for \(C\).

\[0 = \frac{448}{\sqrt{C+1}} - 56\]

\[56 = \frac{448}{\sqrt{C+1}}\]

Multiply by \(\sqrt{C+1}\):

\[56 \sqrt{C+1} = 448\]

Divide by 56:

\[\sqrt{C+1} = \frac{448}{56} = 8\]

Square both sides:

\[C+1 = 8^2 = 64\]

\(C = 63\)

The firm will break even when the cost \(C = \$63\).

Problem 15

Find the demand function \(q=D(x)\) given the following elasticity conditions.
\(E(x)=\frac{x}{220-x},\quad q=210\) when \(x=10\)

Solution:

The formula for elasticity of demand is \(E(x) = -\frac{x}{q} \cdot \frac{dq}{dx}\).

We are given \(E(x)\), so we can set up a differential equation:

\[\frac{x}{220-x} = -\frac{x}{q} \cdot \frac{dq}{dx}\]

We can divide both sides by \(x\) (assuming \(x \neq 0\)):

\[\frac{1}{220-x} = -\frac{1}{q} \cdot \frac{dq}{dx}\]

Now, we separate the variables. Multiply by \(-dx\):

\[\frac{-1}{220-x} dx = \frac{1}{q} dq\]

Integrate both sides:

\[\int \frac{-1}{220-x} dx = \int \frac{1}{q} dq\]

For the left side, let \(u = 220-x\), so \(du = -dx\). The integral becomes \(\int \frac{1}{u} du\).

\[\ln|u| = \ln|q| + C_1\]

Substitute back \(u = 220-x\). Assuming \(x < 220\) and \(q > 0\):

\[\ln(220-x) = \ln(q) + C_1\]

Let's solve for \(q\). First, isolate \(\ln(q)\):

\[\ln(q) = \ln(220-x) - C_1\]

Make both sides the exponent of \(e\):

\[q = e^{\ln(220-x) - C_1} = e^{\ln(220-x)} \cdot e^{-C_1}\]

Let \(C = e^{-C_1}\). This is just a new constant.

\[q = C(220-x)\]

This is the general solution. Now use the condition \(q=210\) when \(x=10\) to find \(C\).

\(210 = C(220 - 10)\)

\(210 = C(210)\)

\(C = 1\)

So the demand function is \(q = 1(220 - x)\), or \(q = 220 - x\).

(Note: This solution assumes \(x < 220\) and \(q > 0\) for the logarithms from the separation of variables to be defined.)

Problem 16

The validity of the Weber-Fechner Law... \(\frac{dR}{dS}=k\cdot\frac{R}{S}\) where k is a positive constant... Find the general solution of this equation.

Solution:

This is a separable differential equation.

\[\frac{dR}{dS}=k\cdot\frac{R}{S}\]

Step 1: Separate the variables.

Divide by \(R\) and multiply by \(dS\).

\[\frac{1}{R} dR = \frac{k}{S} dS\]

Step 2: Integrate both sides.

\[\int \frac{1}{R} dR = \int \frac{k}{S} dS\]

\[\ln|R| = k \int \frac{1}{S} dS\]

\[\ln|R| = k \ln|S| + C_1\]

Step 3: Solve for R.

Using a log property, we can move the \(k\) into the logarithm: \(k \ln|S| = \ln|S^k|\).

\[\ln|R| = \ln|S^k| + C_1\]

Make both sides the exponent of \(e\):

\[e^{\ln|R|} = e^{\ln|S^k| + C_1}\]

\[|R| = e^{\ln|S^k|} \cdot e^{C_1}\]

\[|R| = |S^k| \cdot e^{C_1}\]

Let \(C = \pm e^{C_1}\). This is our new constant.

The general solution is \(R = CS^k\).

Problem 17

Explain the difference between a constant rate of growth and a constant percentage rate of growth.

Solution:

This is about the differential equations that model growth.

Constant rate of growth: This means the population (or amount) \(P\) increases by the same constant amount every time period. For example, adding 100 people every year, regardless of the population size.
The differential equation for this is \(\frac{dP}{dt} = k\), where \(k\) is a constant (e.g., \(k=100\)).

Constant percentage rate of growth: This means the population increases by an amount that is proportional to its current size. For example, growing by 2% of the current population each year. When the population is small, it adds few people. When the population is large, it adds many people. This is "compounded" growth.
The differential equation for this is \(\frac{dP}{dt} = kP\), where \(k\) is the constant percentage rate (e.g., \(k=0.02\)).

Looking at the choices, C is the best description: "For a constant rate of growth, each year the population increases by a constant. For a constant percentage rate of growth, each year the population increases by a constant percentage of the previous year's population."

Problem 18

Let \(f(x,y)=x^{2}-3xy-y^{2}\). Compute \(f(5,0)\) and \(f(5,-5)\).

Solution:

Problem 28

Notice that \(\frac{\sqrt{hw}}{60}\) is just \(S_{orig}\). \[ S_{new} = \sqrt{0.75} \cdot S_{orig} \]

Now, let's calculate \(\sqrt{0.75}\): \[ \sqrt{0.75} \approx 0.866 \]

So, \(S_{new} \approx 0.866 \cdot S_{orig}\).
This means the new surface area is about 86.6% of the original surface area.

The question asks for the percentage *change* (the drop).
Percentage drop = \((1 - 0.866) \times 100\% = 13.4\%\)

This is a 13.4% decrease. The multiple-choice option "A. It drops by approximately 10%" is the closest, but 13.4% is the more precise answer.

Problem 30

Find \(f_{x}\) \(f_{y}\) \(f_{x}(1,2)\) and \(f_{y}(4,-1)\) for the following equation.
\(f(x,y)=\sqrt{x^{2}+y^{2}}\)

Solution:

First, rewrite the function using a power: \(f(x,y) = (x^2 + y^2)^{1/2}\).

Find \(f_x\) (partial derivative with respect to x):

We treat \(y\) as a constant. We must use the Chain Rule.
Let \(u = x^2 + y^2\). Then \(f = u^{1/2}\).
\[f_x = \frac{df}{du} \cdot \frac{\partial u}{\partial x}\]
\[f_x = \left( \frac{1}{2}u^{-1/2} \right) \cdot \left( \frac{\partial}{\partial x}(x^2 + y^2) \right)\]
\[f_x = \left( \frac{1}{2}(x^2+y^2)^{-1/2} \right) \cdot (2x + 0)\]
\[f_x = \frac{1}{2\sqrt{x^2+y^2}} \cdot 2x\]

\[f_x = \frac{x}{\sqrt{x^2+y^2}}\]

Find \(f_y\) (partial derivative with respect to y):

We treat \(x\) as a constant. The process is identical due to symmetry.
\[f_y = \left( \frac{1}{2}(x^2+y^2)^{-1/2} \right) \cdot \left( \frac{\partial}{\partial y}(x^2 + y^2) \right)\]
\[f_y = \left( \frac{1}{2\sqrt{x^2+y^2}} \right) \cdot (0 + 2y)\]

\[f_y = \frac{y}{\sqrt{x^2+y^2}}\]

Find \(f_x(1,2)\):

Plug \(x=1\) and \(y=2\) into the \(f_x\) formula.
\[f_x(1,2) = \frac{1}{\sqrt{1^2+2^2}} = \frac{1}{\sqrt{1+4}} = \frac{1}{\sqrt{5}}\]

Find \(f_y(4,-1)\):

Plug \(x=4\) and \(y=-1\) into the \(f_y\) formula.
\[f_y(4,-1) = \frac{-1}{\sqrt{4^2+(-1)^2}} = \frac{-1}{\sqrt{16+1}} = \frac{-1}{\sqrt{17}}\]

\[f_{yy} = (12x) \cdot [ e^{12xy} \cdot (12x) ] = 144x^2e^{12xy}\]

Problem 41

Find all second order derivatives for \(r(x,y)=\frac{xy}{8x+3y}\)

Solution:

First-order derivatives: (Using the Quotient Rule: \((\frac{f}{g})' = \frac{f'g - fg'}{g^2}\))

\(r_x\): (Let \(f=xy\), \(g=8x+3y\). Differentiate w.r.t. \(x\).)
\[r_x = \frac{(\frac{\partial}{\partial x}(xy)) \cdot (8x+3y) - (xy) \cdot (\frac{\partial}{\partial x}(8x+3y))}{(8x+3y)^2}\]
\[r_x = \frac{(y)(8x+3y) - (xy)(8)}{(8x+3y)^2}\]
\[r_x = \frac{8xy + 3y^2 - 8xy}{(8x+3y)^2} = \frac{3y^2}{(8x+3y)^2}\]

\(r_y\): (Let \(f=xy\), \(g=8x+3y\). Differentiate w.r.t. \(y\).)
\[r_y = \frac{(\frac{\partial}{\partial y}(xy)) \cdot (8x+3y) - (xy) \cdot (\frac{\partial}{\partial y}(8x+3y))}{(8x+3y)^2}\]
\[r_y = \frac{(x)(8x+3y) - (xy)(3)}{(8x+3y)^2}\]
\[r_y = \frac{8x^2 + 3xy - 3xy}{(8x+3y)^2} = \frac{8x^2}{(8x+3y)^2}\]

Second-order derivatives: (Rewrite first-order derivatives for Chain Rule)

\(r_x = 3y^2(8x+3y)^{-2}\)
\(r_y = 8x^2(8x+3y)^{-2}\)

\(r_{xx}\): (Differentiate \(r_x\) w.r.t. \(x\). Treat \(3y^2\) as a constant.)
\[r_{xx} = (3y^2) \cdot \left[ -2(8x+3y)^{-3} \cdot \frac{\partial}{\partial x}(8x+3y) \right]\]
\[r_{xx} = (3y^2) \cdot \left[ -2(8x+3y)^{-3} \cdot (8) \right]\]
\[r_{xx} = -48y^2(8x+3y)^{-3} = \frac{-48y^2}{(8x+3y)^3}\]

\(r_{yy}\): (Differentiate \(r_y\) w.r.t. \(y\). Treat \(8x^2\) as a constant.)
\[r_{yy} = (8x^2) \cdot \left[ -2(8x+3y)^{-3} \cdot \frac{\partial}{\partial y}(8x+3y) \right]\]
\[r_{yy} = (8x^2) \cdot \left[ -2(8x+3y)^{-3} \cdot (3) \right]\]
\[r_{yy} = -48x^2(8x+3y)^{-3} = \frac{-48x^2}{(8x+3y)^3}\]

\(r_{xy}\): (Differentiate \(r_x\) w.r.t. \(y\). Must use Quotient Rule.)
\[r_x = \frac{3y^2}{(8x+3y)^2}\]
\[r_{xy} = \frac{(\frac{\partial}{\partial y}(3y^2)) \cdot (8x+3y)^2 - (3y^2) \cdot (\frac{\partial}{\partial y}(8x+3y)^2)}{((8x+3y)^2)^2}\]
\[r_{xy} = \frac{(6y)(8x+3y)^2 - (3y^2) \cdot (2(8x+3y)^1 \cdot 3)}{(8x+3y)^4}\]
Cancel one \((8x+3y)\) from every term:
\[r_{xy} = \frac{(6y)(8x+3y) - (3y^2)(2 \cdot 3)}{(8x+3y)^3}\]
\[r_{xy} = \frac{48xy + 18y^2 - 18y^2}{(8x+3y)^3}\]
\[r_{xy} = \frac{48xy}{(8x+3y)^3}\]

Since \(r_{yx} = r_{xy}\), we have \(r_{yx} = \frac{48xy}{(8x+3y)^3}\) as well.

Problem 42

A sports company has the following production function...
\(p(x,y)=2000x^{2/5}y^{3/5}\)
(a) Find the number of units produced with 29 units of labor (x) and 1038 units of capital (y).
(b) Find the marginal productivities.
(c) Evaluate the marginal productivities at \(x=29\) and \(y=1038\).
(d) Interpret the meanings...

Solution:

(a) Find p(29, 1038)

\[p(29, 1038) = 2000 \cdot (29)^{2/5} \cdot (1038)^{3/5}\]

Using a calculator:
\((29)^{2/5} = (29^{0.4}) \approx 3.73718\)

\((1038)^{3/5} = (1038^{0.6}) \approx 66.3606\)

\[p(29, 1038) \approx 2000 \cdot (3.73718) \cdot (66.3606) \approx 496,257.9\]

Rounding to the nearest whole number, \(p = 496,258\) units.

(b) Find the marginal productivities

Marginal productivity of labor is \(p_x\).
\[p_x = \frac{\partial}{\partial x}(2000x^{2/5}y^{3/5}) = (2000y^{3/5}) \cdot \frac{d}{dx}(x^{2/5})\]
\[p_x = (2000y^{3/5}) \cdot (\frac{2}{5}x^{2/5 - 1}) = (2000y^{3/5}) \cdot (\frac{2}{5}x^{-3/5})\]
\[p_x = (2000 \cdot \frac{2}{5}) x^{-3/5}y^{3/5} = 800x^{-3/5}y^{3/5}\]

Marginal productivity of capital is \(p_y\).
\[p_y = \frac{\partial}{\partial y}(2000x^{2/5}y^{3/5}) = (2000x^{2/5}) \cdot \frac{d}{dy}(y^{3/5})\]
\[p_y = (2000x^{2/5}) \cdot (\frac{3}{5}y^{3/5 - 1}) = (2000x^{2/5}) \cdot (\frac{3}{5}y^{-2/5})\]
\[p_y = (2000 \cdot \frac{3}{5}) x^{2/5}y^{-2/5} = 1200x^{2/5}y^{-2/5}\]

(c) Evaluate at x=29, y=1038

\[p_x(29, 1038) = 800(29)^{-3/5}(1038)^{3/5}\]
\((29)^{-3/5} = (29^{-0.6}) \approx 0.12885\)

\((1038)^{3/5} \approx 66.3606\)

\[p_x \approx 800 \cdot (0.12885) \cdot (66.3606) \approx 6845.0\]

Rounding to the nearest whole number, \(p_x(29, 1038) = 6845\).


\[p_y(29, 1038) = 1200(29)^{2/5}(1038)^{-2/5}\]
\((29)^{2/5} \approx 3.73718\)

\((1038)^{-2/5} = (1038^{-0.4}) \approx 0.06401\)

\[p_y \approx 1200 \cdot (3.73718) \cdot (0.06401) \approx 287.0\]

Rounding to the nearest whole number, \(p_y(29, 1038) = 287\).

(d) Interpret the meanings

\(p_x = 6845\) (marginal productivity of labor): If capital is held fixed at 1038 units, adding one more unit of labor (going from 29 to 30) will increase production by approximately 6,845 units.

\(p_y = 287\) (marginal productivity of capital): If labor is held fixed at 29 units, adding one more unit of capital (going from 1038 to 1039) will increase production by approximately 287 units.

Problem 44

The temperature-humidity index, \(T_{h}\)... is given by \(T_{h}=1.98T-1.09(1-H)(T-58)-56.9,\)...
Find the temperature-humidity index if \(T=78^{\circ}\) and \(H=62\%\).

Solution:

This is a function evaluation. We must plug in \(T = 78\) and \(H = 0.62\) (since H is a decimal).

\[T_h = 1.98(78) - 1.09(1 - 0.62)(78 - 58) - 56.9\]

Solve step-by-step:

\(1.98(78) = 154.44\)

\(1 - 0.62 = 0.38\)

\(78 - 58 = 20\)

Substitute these back in:

\[T_h = 154.44 - 1.09(0.38)(20) - 56.9\]

Now, calculate the middle term:

\[1.09 \times 0.38 \times 20 = 1.09 \times 7.6 = 8.284\]

Substitute this back in:

\[T_h = 154.44 - 8.284 - 56.9\]

\[T_h = 146.156 - 56.9\]

\[T_h = 89.256\]

Rounding to one decimal place, the temperature-humidity index is \(89.3\).
