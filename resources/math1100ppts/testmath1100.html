Section 1

Differential Equations

What you should understand from this section
  • A solution to a differential equation is a function, not just a number.
  • Separation of variables turns some differential equations into two integrals.
  • An initial condition selects one particular function from a family of possible solutions.

A differential equation is an equation involving an unknown function and one or more of its derivatives. In this section, the unknown function is usually \(y\), and the derivative describes how \(y\) changes.

The main shift in thinking is that solving the equation means finding a function. If an initial condition is provided, the answer should be a particular function that satisfies both the differential equation and the initial condition.

Key Idea: Separation of Variables

A first-order differential equation is separable when it can be rearranged so that all terms involving \(y\) and \(dy\) appear on one side and all terms involving \(x\) and \(dx\) appear on the other side. A typical form is

\[ g(y)\,dy=h(x)\,dx. \]

After separation, integrate both sides and use any initial condition to determine the constant.

Common Mistake

Do not integrate only the right-hand side and leave \(y\) unchanged. In a separable equation, both sides represent accumulated change, so both sides must be integrated after the variables have been separated.

Interactive: Continuous Growth Modeler

Observe how changing \(k\) changes the direction and steepness of exponential change.

For \(y=Ce^{kt}\), the derivative is \(dy/dt=kCe^{kt}=ky\). The parameter \(C\) controls the initial amount, and \(k\) controls the continuous proportional rate.

Interpret what changed: when \(k>0\), \(y\) grows and \(y'\) is positive; when \(k<0\), \(y\) decays and \(y'\) is negative; when \(k=0\), \(y\) is constant.

Worked Example: Initial Value Problem

Solve \(y'=\dfrac{6x+1}{2y}\), with \(y(2)=3\).

Solution

Step 1: Separate variables
\[ \frac{dy}{dx}=\frac{6x+1}{2y} \quad\Longrightarrow\quad 2y\,dy=(6x+1)\,dx. \]
Step 2: Integrate both sides
\[ \int 2y\,dy=\int (6x+1)\,dx \quad\Longrightarrow\quad y^2=3x^2+x+C. \]
Step 3: Use the initial condition
\[ 3^2=3(2)^2+2+C \quad\Longrightarrow\quad 9=14+C \quad\Longrightarrow\quad C=-5. \]
Step 4: Solve for the particular function
\[ y^2=3x^2+x-5. \]

Since the initial value \(y(2)=3\) is positive, the particular solution near \(x=2\) uses the positive square root.

Conclusion
\[ \boxed{y=\sqrt{3x^2+x-5}}. \]

The solution is the positive branch of the square-root function determined by the initial value.

Check: Differentiating gives \(2yy'=6x+1\), so \(y'=(6x+1)/(2y)\). Also \(y(2)=\sqrt{12+2-5}=3\).

Applied Example: Weber-Fechner Law

Why this matters: this model describes situations where perception changes by equal amounts when stimulus intensity changes by equal ratios.

The Weber-Fechner model writes the response rate as

\[ \frac{dR}{dS}=\frac{k}{S}, \]

where \(S>0\) is stimulus intensity and \(k\) is a constant.

Solution and Interpretation

Step 1: Separate and integrate
\[ dR=\frac{k}{S}\,dS \quad\Longrightarrow\quad R(S)=k\ln S+C. \]

The logarithm requires \(S>0\), which is consistent with stimulus intensity being positive.

Step 2: Write the reference form

If a reference stimulus \(S_0>0\) has response \(R_0\), then \(R_0=k\ln S_0+C\). Subtracting gives

\[ \boxed{R(S)=R_0+k\ln\!\left(\frac{S}{S_0}\right)}. \]
Interpretation

The logarithmic form means equal multiplicative changes in stimulus intensity produce equal additive changes in response. Doubling from \(S_0\) to \(2S_0\) changes the response by \(k\ln 2\), and doubling again from \(2S_0\) to \(4S_0\) changes it by another \(k\ln 2\).

Checkpoint: differentiating \(R(S)=k\ln S+C\) gives \(dR/dS=k/S\).

Section 2

Functions of 3+ Variables

What you should understand from this section
  • Functions can have three, four, or more independent input variables.
  • Evaluating a multivariable function requires substituting a value for each variable.
  • Partial derivatives work identically: treat all other variables as constants and differentiate with respect to one specific variable.

Worked Example: Evaluating a 3-Variable Function

Evaluate \(f(x,y,z)=4x-6y^{2}+7z^{3}-6\) at \(x=2\), \(y=-1\), and \(z=2\).

Solution

Step 1: Substitute the given values
\[ f(2,-1,2)=4(2)-6(-1)^{2}+7(2)^{3}-6 \]
Step 2: Simplify exponents and multiply
\[ f(2,-1,2)=8-6(1)+7(8)-6 = 8-6+56-6 \]
Step 3: Compute the final sum
\[ f(2,-1,2)=52 \]
Conclusion

The value of the function is \(\boxed{52}\).

Worked Example: Partial Derivatives of a 4-Variable Function

Find the partial derivatives \(P_w\), \(P_r\), \(P_s\), and \(P_t\) for \(P(w,r,s,t)=w^2r + 3rs^2 - 5st^3 + 10\).

Solution

With respect to \(w\)

Treat \(r, s, t\) as constants.

\[ P_w = 2wr \]
With respect to \(r\)

Treat \(w, s, t\) as constants. Both the first and second terms contain \(r\).

\[ P_r = w^2 + 3s^2 \]
With respect to \(s\)

Treat \(w, r, t\) as constants.

\[ P_s = 6rs - 5t^3 \]
With respect to \(t\)

Treat \(w, r, s\) as constants.

\[ P_t = -15st^2 \]
Conclusion
\[ \boxed{P_w = 2wr}, \quad \boxed{P_r = w^2 + 3s^2}, \quad \boxed{P_s = 6rs - 5t^3}, \quad \boxed{P_t = -15st^2} \]

Section 3

Partial Derivatives

What you should understand from this section
  • A function \(f(x,y)\) has two independent input variables.
  • A partial derivative changes one input while holding the other fixed.
  • Domain restrictions still matter, especially for square roots, logarithms, denominators, and applied quantities.

Key Idea: Two Inputs, One Output

A function of two variables has the form \(z=f(x,y)\). Its input is an ordered pair, and its graph is often a surface in three-dimensional space.

Domain restrictions can come from roots, logarithms, denominators, or the applied context. For example, \(\sqrt{y-8x}\) requires \(y-8x\ge 0\), and a production model with labor and capital typically requires positive inputs.

Key Idea: Holding One Variable Constant

A partial derivative measures how the output changes with respect to one input while the other input is held constant. When computing \(f_x\), treat \(y\) as a constant. When computing \(f_y\), treat \(x\) as a constant.

Common Mistake

Do not treat both variables as changing when computing a partial derivative. For \(f_x\), every expression involving only \(y\) is constant with respect to \(x\). For \(f_y\), every expression involving only \(x\) is constant with respect to \(y\).

Partial Derivative Notation

Type Notation Meaning Operation
First-order partial with respect to \(x\) \(f_x\), \(\dfrac{\partial f}{\partial x}\) Rate of change of \(f\) as \(x\) changes while \(y\) is fixed. Differentiate with respect to \(x\); treat \(y\) as constant.
First-order partial with respect to \(y\) \(f_y\), \(\dfrac{\partial f}{\partial y}\) Rate of change of \(f\) as \(y\) changes while \(x\) is fixed. Differentiate with respect to \(y\); treat \(x\) as constant.
Second-order pure partial in \(x\) \(f_{xx}\), \(\dfrac{\partial^2 f}{\partial x^2}\) How \(f_x\) changes as \(x\) changes while \(y\) is fixed. Differentiate \(f_x\) with respect to \(x\).
Second-order pure partial in \(y\) \(f_{yy}\), \(\dfrac{\partial^2 f}{\partial y^2}\) How \(f_y\) changes as \(y\) changes while \(x\) is fixed. Differentiate \(f_y\) with respect to \(y\).
Second-order mixed partial \(xy\) \(f_{xy}\), \(\dfrac{\partial^2 f}{\partial y\,\partial x}\) How \(f_x\) changes as \(y\) changes. Differentiate first with respect to \(x\), then with respect to \(y\).
Second-order mixed partial \(yx\) \(f_{yx}\), \(\dfrac{\partial^2 f}{\partial x\,\partial y}\) How \(f_y\) changes as \(x\) changes. Differentiate first with respect to \(y\), then with respect to \(x\).

Interactive: Slices of \(z=4x^2+y^2\)

Observe that a partial derivative comes from a one-variable slice of a two-variable surface.

The selected curve shows the slice produced by holding one input constant. The selected point lies on both the surface and the slice. A tangent trace is included along the selected slice direction.

Slice direction

Interpret what changed: switching the held variable changes which one-variable curve is being studied and therefore which partial derivative is represented.

Worked Symbolic Example

Find \(f_{xx}\) and \(f_{xy}\) for \(f(x,y)=e^{10xy}\).

Solution

Step 1: Compute \(f_x\)

Treat \(y\) as constant and use the chain rule.

\[ f_x=10y e^{10xy}. \]
Step 2: Differentiate \(f_x\) with respect to \(x\)
\[ f_{xx}=10y\left(10y e^{10xy}\right) =100y^2 e^{10xy}. \]
Step 3: Differentiate \(f_x\) with respect to \(y\)

Use the product rule because \(10y\) and \(e^{10xy}\) both depend on \(y\).

\[ f_{xy}=10 e^{10xy}+10y(10x e^{10xy}) =(10+100xy)e^{10xy}. \]
Conclusion
\[ \boxed{f_{xx}=100y^2 e^{10xy}},\qquad \boxed{f_{xy}=(10+100xy)e^{10xy}}. \]
Structure check: \(f_{xy}\) uses the product rule because \(10y\) and \(e^{10xy}\) both depend on \(y\).

Applied Example: Cobb-Douglas Production

Why this matters: a partial derivative in a production model estimates the marginal effect of changing one input while holding the other input fixed.

Let \(P(x,y)=2100x^{3/5}y^{2/5}\), where \(x>0\) is labor and \(y>0\) is capital. Compute \(P_x\) and interpret it.

Solution and Interpretation

Step 1: State the domain for the applied model

The formula uses fractional powers of labor and capital, so the applied model assumes \(x>0\) and \(y>0\).

Step 2: Hold capital fixed and differentiate with respect to labor
\[ P_x=2100\cdot \frac{3}{5}x^{-2/5}y^{2/5} =1260x^{-2/5}y^{2/5}. \]
Conclusion
\[ \boxed{P_x=1260x^{-2/5}y^{2/5}}. \]

Equivalently, for positive \(x\) and \(y\), \(P_x=1260\left(\dfrac{y}{x}\right)^{2/5}\).

Interpretation

The partial derivative \(P_x\) is the marginal productivity of labor: the approximate increase in production from one additional unit of labor, holding capital fixed. Since the exponent of \(x\) in \(P_x\) is negative, this marginal productivity decreases as labor increases while capital remains constant.

Section 4

Financial Amortization & Tabular Data

What you should understand from this section
  • Reading 2D financial tables to identify mortgage or loan constants.
  • Calculating monthly payments and total overall costs based on tabulated factors.
  • Visualizing the relationship between principal paid and interest paid over a term.

Applied Example: Reading Loan Tables

A $13,000 loan is offered from a bank. Use the provided loan factor table to calculate the monthly payment and the total overall cost if the loan has an APR of 6.0% and a term of 4 years.

APR 2 Year Term 3 Year Term 4 Year Term 5 Year Term
5.0% $43.87 $29.97 $23.03 $18.87
6.0% $44.32 $30.42 $23.49 $19.33
7.0% $44.77 $30.88 $23.95 $19.80

Note: Values in the table represent the monthly payment per $1,000 borrowed.

Solution

Step 1: Identify the factor from the table

Find the intersection of the 6.0% APR row and the 4 Year Term column. The factor is $23.49 per $1,000 borrowed.

Step 2: Calculate the monthly payment

Divide the total loan principal by 1,000 and multiply by the factor.

\[ \text{Monthly Payment} = \frac{13,000}{1,000} \times 23.49 = 13 \times 23.49 = 305.37 \]
Step 3: Calculate the total overall cost

Multiply the monthly payment by the total number of months in the term (4 years \(\times\) 12 months/year = 48 months).

\[ \text{Total Cost} = 305.37 \times 48 = 14,657.76 \]
Conclusion

The monthly payment is \(\boxed{\$305.37}\) and the total overall cost of the loan is \(\boxed{\$14,657.76}\).

Section 5

HW-10 Target Solutions Archive

What you should use this archive for
  • Review the method before checking the final answer.
  • Notice which problems require integration, differentiation, domain reasoning, or interpretation.
  • Use the checkpoints to verify that the final expression satisfies the original condition.

Each item below includes a complete solution path and a brief check where useful. Open the items as needed.

1. Find the particular solution for \(y'=8x^7\); \(y(0)=3\).
Step 1: Integrate the derivative
\[ y=\int 8x^7\,dx=x^8+C. \]
Step 2: Apply the initial condition
\[ 3=y(0)=0^8+C \quad\Longrightarrow\quad C=3. \]
Conclusion
\[ \boxed{y=x^8+3}. \]

The particular solution is the antiderivative whose graph passes through \((0,3)\).

Check: \(y'=8x^7\), and \(y(0)=3\).
2. Solve \(\dfrac{dy}{dx}=12x^{11}y\).
Step 1: Separate variables
\[ \frac{1}{y}\,dy=12x^{11}\,dx. \]
Step 2: Integrate both sides
\[ \ln|y|=x^{12}+C. \]

The logarithm appears because \(\int (1/y)\,dy=\ln|y|\) for \(y\ne 0\).

Step 3: Solve for \(y\)
\[ |y|=e^C e^{x^{12}}. \]

Absorb the sign and \(e^C\) into a single arbitrary constant.

Conclusion
\[ \boxed{y=Ce^{x^{12}}}. \]
Check: \(y'=C e^{x^{12}}\cdot 12x^{11}=12x^{11}y\). The zero solution is included when \(C=0\).
3. Show that \(y=x\ln x+4x-2\) is a solution of \(y''-\dfrac{1}{x}=0\).
Step 1: State the domain

The logarithm requires \(x>0\), so the verification is on the interval \(x>0\).

Step 2: Differentiate once
\[ y'=\frac{d}{dx}(x\ln x)+\frac{d}{dx}(4x)-\frac{d}{dx}(2) =(\ln x+1)+4 =\ln x+5. \]
Step 3: Differentiate again
\[ y''=\frac{1}{x}. \]
Step 4: Substitute into the differential equation
\[ y''-\frac{1}{x}=\frac{1}{x}-\frac{1}{x}=0. \]
Conclusion

Thus \(y=x\ln x+4x-2\) is a solution on \(x>0\).

4. Debra deposits \(\$200\) at \(3.88\%\) compounded continuously. Give the DE, find \(A(t)\), evaluate \(A(4)\), and \(A'(4)\).
Step 1: Write the continuous compounding model

Continuous compounding is modeled by \(dA/dt=rA\). Here \(r=0.0388\) and \(A(0)=200\), so

\[ \frac{dA}{dt}=0.0388A,\qquad A(0)=200. \]
Step 2: Solve the differential equation
\[ A(t)=200e^{0.0388t}. \]
Step 3: Evaluate the account balance after 4 years
\[ A(4)=200e^{0.1552}\approx 233.58. \]
Step 4: Evaluate the instantaneous growth rate
\[ A'(4)=0.0388A(4)=0.0388(200e^{0.1552})\approx 9.06. \]
Conclusion

The model is \(\boxed{A'(t)=0.0388A(t),\ A(0)=200}\), with \(\boxed{A(t)=200e^{0.0388t}}\). After 4 years the account contains about \(\boxed{\$233.58}\), and it is growing at about \(\boxed{\$9.06}\) per year.

5. Find the demand function given \(E(x)=\dfrac{x}{190-x}\) and \(q=160\) when \(x=30\).
Step 1: Use the elasticity convention

Use the standard price elasticity convention \(E(x)=-xq'(x)/q(x)\). The denominator \(190-x\) requires \(x\ne 190\); the given data use \(x=30\).

\[ -\frac{xq'}{q}=\frac{x}{190-x}. \]
Step 2: Separate variables

For \(x>0\), divide by \(x\):

\[ -\frac{q'}{q}=\frac{1}{190-x} \quad\Longrightarrow\quad \frac{q'}{q}=-\frac{1}{190-x}. \] \[ \frac{1}{q}\,dq=-\frac{1}{190-x}\,dx. \]
Step 3: Integrate and solve for \(q\)
\[ \ln|q|=\ln|190-x|+C. \]

For a positive demand quantity in the relevant interval \(x<190\), write

\[ q=C(190-x). \]
Step 4: Use the demand data
\[ 160=C(190-30)=160C \quad\Longrightarrow\quad C=1. \]
Conclusion
\[ \boxed{q=190-x}. \]

The demand quantity decreases linearly with price in this model.

Check: \(q'=-1\), so \(-xq'/q=x/(190-x)\), matching the given elasticity.
6. Let \(f(x,y)=x^2-3xy-y^2\). Compute \(f(5,-7)\).
Step 1: Substitute the ordered pair
\[ f(5,-7)=5^2-3(5)(-7)-(-7)^2. \]
Step 2: Simplify carefully
\[ f(5,-7)=25+105-49=81. \]
Conclusion

The function value is \(\boxed{81}\).

7. Determine the domain of \(f(x,y)=\sqrt{y-8x}\). State the inequality.
Step 1: Identify the restriction

A square root is real when its radicand is nonnegative.

\[ y-8x\ge 0. \]
Step 2: State the domain
\[ \boxed{\{(x,y):y-8x\ge 0\}} \quad\text{or equivalently}\quad \boxed{y\ge 8x}. \]
Interpretation

Geometrically, this is the line \(y=8x\) and the half-plane above it.

8. Mosteller formula \(S(h,w)=\dfrac{\sqrt{hw}}{60}\). If weight drops by \(20\%\), by what percentage does surface area drop?
Step 1: Translate the percent change

Assume height \(h\) remains fixed and weight changes from \(w\) to \(0.8w\). The original surface area is

\[ S_{\text{old}}=\frac{\sqrt{hw}}{60}. \]
Step 2: Compare new surface area to old surface area
\[ S_{\text{new}}=\frac{\sqrt{h(0.8w)}}{60} =\sqrt{0.8}\,\frac{\sqrt{hw}}{60} =\sqrt{0.8}\,S_{\text{old}}. \]
Step 3: Convert the ratio to a percentage drop
\[ (1-\sqrt{0.8})\cdot 100\% \approx (1-0.894427)\cdot 100\% \approx 10.56\%. \]
Conclusion

The surface area drops by about \(\boxed{10.56\%}\), not by \(20\%\), because surface area depends on the square root of weight.

9. Find \(f_x\) and \(f_y\) for \(f(x,y)=y\ln(9x+4y)\).
Step 1: State the domain restriction

The logarithm requires \(9x+4y>0\).

Step 2: Compute \(f_x\)

Hold \(y\) constant.

\[ f_x=y\cdot \frac{1}{9x+4y}\cdot 9 =\frac{9y}{9x+4y}. \]
Step 3: Compute \(f_y\)

Use the product rule because both \(y\) and \(\ln(9x+4y)\) depend on \(y\).

\[ f_y=1\cdot \ln(9x+4y)+y\cdot \frac{1}{9x+4y}\cdot 4. \] \[ f_y=\ln(9x+4y)+\frac{4y}{9x+4y}. \]
Conclusion
\[ \boxed{f_x=\frac{9y}{9x+4y}},\qquad \boxed{f_y=\ln(9x+4y)+\frac{4y}{9x+4y}}. \]
10. Find \(r_{xx}\) for \(r(x,y)=\dfrac{xy}{5x+4y}\). Show the quotient rule carefully.
Step 1: State the domain and quotient-rule pieces

The denominator requires \(5x+4y\ne 0\). When differentiating with respect to \(x\), treat \(y\) as constant. Let

\[ N=xy,\qquad D=5x+4y. \]
Step 2: Compute \(r_x\) using the quotient rule

Here \(N_x=y\) and \(D_x=5\). Therefore

\[ r_x=\frac{N_xD-ND_x}{D^2} =\frac{y(5x+4y)-xy(5)}{(5x+4y)^2}. \]

Simplify the numerator:

\[ y(5x+4y)-5xy=5xy+4y^2-5xy=4y^2. \] \[ r_x=\frac{4y^2}{(5x+4y)^2}. \]
Step 3: Differentiate \(r_x\) with respect to \(x\)

Since \(4y^2\) is constant with respect to \(x\), write \(r_x=4y^2(5x+4y)^{-2}\):

\[ r_{xx}=4y^2\left[-2(5x+4y)^{-3}\cdot 5\right]. \]
Conclusion
\[ \boxed{r_{xx}=-\frac{40y^2}{(5x+4y)^3}}. \]
Quotient-rule check for the second derivative: \(\dfrac{d}{dx}\left(\dfrac{4y^2}{D^2}\right)=\dfrac{0\cdot D^2-4y^2(2D)(5)}{D^4}=-40y^2/D^3\).
