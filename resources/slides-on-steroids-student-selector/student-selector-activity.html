Slides on Steroids

Student Selector — Active Learning Activity

What this is: a projector-friendly, UDL-first activity: fair random selector, question bank, and a big visual timer — all in one slide.

Code licensed MIT · Deck content CC BY 4.0 · Please attribute “Slides on Steroids — Kodipelly (UMSL)”

How to Use

  • Roster: Edit the DUMMY_ROSTER array in the script (or replace with ACTUAL_ROSTER).
  • Prompt bank: Edit QUESTION_BANK with your questions.
  • Run it live: Click Start → timer counts down → spinner reveals student → Spin Again to re-select without changing the question.
  • Accessibility: Big type, high contrast, keyboard focus, no pop-ups. Print-safe from browser (enable background graphics).
