Hypothesis Testing in R (R Code Examples)

This interactive resource presents one-sample and two-sample hypothesis testing examples in R. It covers raw-data and summary statistics examples for Z-tests, T-tests, pooled and unpooled (Welch's) two-sample tests, paired tests, and non-parametric tests. Designed especially for beginners in R-programming in an introductory probability and statistics course, it guides students step-by-step through each test's implementation.
